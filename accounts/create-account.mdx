### Body
<ParamField body="balance_id" type="string" required>
  The ID of an existing balance to link to this account. The account will inherit currency, ledger, and other properties from this balance. You can use the same balance_id for multiple accounts.
</ParamField>
<ParamField body="identity_id" type="string" required>
  The ID of the identity (individual or organization) that owns the account. You can use the same identity_id for multiple accounts.
</ParamField>
<ParamField body="bank_name" type="string" required>
  The name of the bank where the account is held.
</ParamField>
<ParamField body="number" type="string" required>
  The account number. Can be auto-generated if external account generation is enabled.
</ParamField>
<ParamField body="meta_data" type="object">
  Custom metadata for the account. Can include any additional information you want to store.
</ParamField>

### Response
<ResponseField name="account_id" type="string">
  Unique identifier for the created account.
</ResponseField>
<ResponseField name="name" type="string">
  The account name. Auto-generated based on the linked identity if not provided.
</ResponseField>
<ResponseField name="number" type="string">
  The account number.
</ResponseField>
<ResponseField name="bank_name" type="string">
  The name of the bank.
</ResponseField>
<ResponseField name="currency" type="string">
  The account currency.
</ResponseField>
<ResponseField name="balance_id" type="string">
  The ID of the linked balance.
</ResponseField>
<ResponseField name="identity_id" type="string">
  The ID of the linked identity.
</ResponseField>
<ResponseField name="ledger_id" type="string">
  The ID of the linked ledger.
</ResponseField>
<ResponseField name="created_at" type="timestamp">
  ISO 8601 formatted timestamp of when the account was created.
</ResponseField>
<ResponseField name="meta_data" type="object">
  Custom metadata associated with the account.
</ResponseField>

<RequestExample>
```bash
curl --request POST \
  --url http://localhost:5001/accounts \
  --header 'X-blnk-key: <api-key>' \
  --header 'Content-Type: application/json' \
  --data '{
    "balance_id": "bal_123456789",
    "identity_id": "id_123456789",
    "bank_name": "Blnk Bank",
    "number": "1234567890",
    "meta_data": {
      "account_type": "checking",
      "branch_code": "001"
    }
  }'
```
</RequestExample>

<Tip>
**Multiple Accounts Example**: You can create additional accounts for the same identity and balance by using the same `balance_id` and `identity_id` with different account numbers:

```bash
curl --request POST \
  --url http://localhost:5001/accounts \
  --header 'X-blnk-key: <api-key>' \
  --header 'Content-Type: application/json' \
  --data '{
    "balance_id": "bal_123456789",
    "identity_id": "id_123456789",
    "bank_name": "Blnk Bank",
    "number": "9876543210",
    "meta_data": {
      "account_type": "savings",
      "branch_code": "002"
    }
  }'
```
</Tip>
