## Installing Skills

<Steps>
<Step title="Add Skills">
From your project (or any directory where you use the LLM), run:

```bash Install Blnk skills wrap
npx skills add https://docs.blnkfinance.com
```

The CLI detects tools such as Cursor and Claude Code, and installs the skills for you. Confirm when prompted if you want them on more than one tool.
</Step>

<Step title="Confirm they loaded">
To use the skill, start with `/blnk` and ask any question. That is the root entry point; it routes to the other Blnk skills for architecture, money movement, and implementation.
</Step>

<Step title="Connect the Docs MCP">
To improve the LLM's knowledge base with live Docs lookup, connect the Docs MCP in your LLM tool:

```text MCP endpoint
https://docs.blnkfinance.com/mcp
```
</Step>
</Steps>

## Try it: build a simple wallet

Open a chat in your project and run through these steps.

<Steps>
<Step title="Share this sample spec">
Paste:

```text Sample wallet spec wrap
/blnk

Build this on Blnk:

# Customer wallets (POC)

- Each customer gets a USD wallet
- Deposit from a bank into the wallet
- Transfer between two customers

Design the money movement first, then implement it in this repo.
```
</Step>

<Step title="Answer follow-up questions">
The agent asks a few clarifying questions. Answer them so it can lock the design.
</Step>

<Step title="Review the map">
The agent posts maps-tool JSON in chat and saves it under `.blnk_context/`.

To view it:

1. Open [map.blnkfinance.com](https://map.blnkfinance.com).
2. Click **Menu → Import**.
3. Paste the JSON and load it.

If anything is wrong, say so in chat and wait for an updated JSON. When it matches, continue.
</Step>

<Step title="Ask it to implement">
```text wrap
Map looks good. Implement it with the TypeScript SDK.
Use a Blnk Cloud sandbox unless we are self-hosting.
```
</Step>

<Step title="Start the app">
Once done, you can start your app and should have a running wallet flow against Blnk.
</Step>
</Steps>
