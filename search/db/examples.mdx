## High-value applied transactions

Retrieve all applied transactions above $10,000, sorted by amount:

<CodeGroup>
```bash cURL wrap
curl -X POST "https://YOUR_BLNK_INSTANCE_URL/transactions/filter" \
  -H "X-Blnk-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "filters": [
      { "field": "status", "operator": "eq", "value": "APPLIED" },
      { "field": "amount", "operator": "gte", "value": 10000 }
    ],
    "sort_by": "amount",
    "sort_order": "desc",
    "limit": 50
  }'
```

```typescript TypeScript wrap
const response = await blnk.Search.filter(
  {
    filters: [
      {
        field: 'status',
        operator: 'eq',
        value: 'APPLIED',
      },
      {
        field: 'amount',
        operator: 'gte',
        value: 10000,
      },
    ],
    sort_by: 'amount',
    sort_order: 'desc',
    limit: 50,
  },
  'transactions',
);
```

```go Go wrap
results, resp, err := client.Transaction.Filter(blnkgo.FilterParams{
    Filters: []blnkgo.Filter{
        {Field: "status", Operator: blnkgo.OpEqual, Value: "APPLIED"},
        {Field: "amount", Operator: blnkgo.OpGTE, Value: 10000},
    },
    SortBy: "amount",
    SortOrder: "desc",
    Limit: 50,
})
```

```python Python wrap
response = blnk.search.filter(
  {
    "filters": [
      {
        "field": "status",
        "operator": "eq",
        "value": "APPLIED",
      },
      {
        "field": "amount",
        "operator": "gte",
        "value": 10000,
      },
    ],
    "sort_by": "amount",
    "sort_order": "desc",
    "limit": 50,
  },
  "transactions",
)
```

```java Java wrap
ApiResponse<JsonNode> response = blnk.search().filter(
    FilterParams.create()
        .filters(List.of(
            FilterCondition.create()
            .field("status")
            .operator("eq")
            .value("APPLIED"),
            FilterCondition.create()
            .field("amount")
            .operator("gte")
            .value(10000)
        ))
        .sortBy("amount")
        .sortOrder("desc")
        .limit(50),
    "transactions");
```
</CodeGroup>

```json Response wrap
{
  "data": [
    {
      "transaction_id": "txn_6164573b-6cc8-45a4-ad2e-7b4ba6a60f7d",
      "amount": 15000,
      "currency": "USD",
      "status": "APPLIED",
      "source": "bln_28edb3e5-c168-4127-a1c4-16274e7a28d3",
      "destination": "bln_ebcd230f-6265-4d4a-a4ca-45974c47f746",
      "created_at": "2024-01-15T10:30:00Z"
    }
  ]
}
```

## USD balances with positive balance

Find all USD balances with a positive balance and include the total count:

<CodeGroup>
```bash cURL wrap
curl -X POST "https://YOUR_BLNK_INSTANCE_URL/balances/filter" \
  -H "X-Blnk-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "filters": [
      { "field": "currency", "operator": "eq", "value": "USD" },
      { "field": "balance", "operator": "gt", "value": 0 }
    ],
    "include_count": true
  }'
```

```typescript TypeScript wrap
const response = await blnk.Search.filter(
  {
    filters: [
      {
        field: 'currency',
        operator: 'eq',
        value: 'USD',
      },
      {
        field: 'balance',
        operator: 'gt',
        value: 0,
      },
    ],
    include_count: true,
  },
  'balances',
);
```

```go Go wrap
results, resp, err := client.LedgerBalance.Filter(blnkgo.FilterParams{
    Filters: []blnkgo.Filter{
        {Field: "currency", Operator: blnkgo.OpEqual, Value: "USD"},
        {Field: "balance", Operator: blnkgo.OpGT, Value: 0},
    },
    IncludeCount: true,
})
```

```python Python wrap
response = blnk.search.filter(
  {
    "filters": [
      {
        "field": "currency",
        "operator": "eq",
        "value": "USD",
      },
      {
        "field": "balance",
        "operator": "gt",
        "value": 0,
      },
    ],
    "include_count": True,
  },
  "balances",
)
```

```java Java wrap
ApiResponse<JsonNode> response = blnk.search().filter(
    FilterParams.create()
        .filters(List.of(
            FilterCondition.create()
            .field("currency")
            .operator("eq")
            .value("USD"),
            FilterCondition.create()
            .field("balance")
            .operator("gt")
            .value(0)

        .includeCount(true),
    "balances");
```
</CodeGroup>

```json Response wrap

      "debit_balance": 25000,
      "ledger_id": "ldg_073f7ffe-9dfd-42ce-aa50-d1dca1788adc"
```

## Multiple currencies with pattern search

Find all ledgers with "savings" in the name, sorted alphabetically:

<CodeGroup>
```bash cURL wrap
curl -X POST "https://YOUR_BLNK_INSTANCE_URL/ledgers/filter" \
  -H "X-Blnk-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "filters": [
      { "field": "name", "operator": "ilike", "value": "%savings%" }
    ],
    "sort_by": "name",
    "sort_order": "asc"
  }'
```

```typescript TypeScript wrap
const response = await blnk.Search.filter(
  {
    filters: [
      {
        field: 'name',
        operator: 'ilike',
        value: '%savings%',
      },
    ],
    sort_by: 'name',
    sort_order: 'asc',
  },
  'ledgers',
);
```

```go Go wrap
results, resp, err := client.Ledger.Filter(blnkgo.FilterParams{
    Filters: []blnkgo.Filter{
        {Field: "name", Operator: blnkgo.OpILike, Value: "%savings%"},

    SortBy: "name",
    SortOrder: "asc",
})
```

```python Python wrap
response = blnk.search.filter(
  {
    "filters": [
      {
        "field": "name",
```
