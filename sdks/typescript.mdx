## Install

Create a Node.js project and install the SDK. You need [Node.js 18 or later](https://nodejs.org/en/download).

```bash bash wrap
npm init -y
npm install @blnkfinance/blnk-typescript
npm install -D tsx typescript
```

Already have the SDK installed? Upgrade to the latest published package:

```bash bash wrap
npm install @blnkfinance/blnk-typescript@latest
```

Check the installed version:

```bash bash wrap
npm ls @blnkfinance/blnk-typescript
```
