Use `client.ApiKeys.Delete` to revoke an [API key](/advanced/secure-blnk) by ID.

<Steps>
<Step title="Call the method">

```go client.ApiKeys.Delete wrap
resp, err := client.ApiKeys.Delete(
	"api_key_879f0ecb-e29f-4137-801b-1048366381db",
	&blnkgo.DeleteApiKeysOptions{
		Owner: "merchant_a",
	},
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `apiKeyID` | `string` | ID of the API key to revoke. |
| `options.`<br />`Owner` | `string` | Owner of the key. Required when authenticating with the master key. |

</Step>

<Step title="Confirm the outcome">

On success, Core returns `204 No Content`. Check `resp.StatusCode`. The key can no longer authenticate requests.

</Step>

<Step title="Response">

```json 204 No Content wrap
{}
```

Returns `204 No Content` on success with an empty body. `Delete` returns `(*http.Response, error)` only — there is no decoded response struct.

</Step>
</Steps>
