Use `client.Reconciliation.DeleteMatchingRule` to delete a matching rule.

<Steps>
<Step title="Call the method">

```go client.Reconciliation.DeleteMatchingRule wrap
deleted, resp, err := client.Reconciliation.DeleteMatchingRule("rule_890bdbc4-467f-4670-8424-5667e41daf29")
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `ruleID` | `string` | ID of the matching rule to delete. |

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "message": "Matching rule deleted successfully"
}
```

#### Go response (`*blnkgo.DeleteMatchingRuleResp`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `Message` | `string` | Confirmation that the rule was deleted. |

</Step>
</Steps>
