Use `client.Reconciliation.UpdateMatchingRule` to update a matching rule.

<Steps>
<Step title="Call the method">

```go client.Reconciliation.UpdateMatchingRule wrap
updatedRule, resp, err := client.Reconciliation.UpdateMatchingRule(
	"rule_890bdbc4-467f-4670-8424-5667e41daf29",
	blnkgo.Matcher{
		Name: "Amount match (updated)",
		Description: "Match by amount with 2% drift",
		Criteria: []blnkgo.Criteria{
			{
				Field: blnkgo.CriteriaFieldAmount,
				Operator: blnkgo.ReconciliationOperatorEquals,
				AllowableDrift: 0.02,
			},
			{
				Field: blnkgo.CriteriaFieldCurrency,
				Operator: blnkgo.ReconciliationOperatorEquals,
			},
		},
	},
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `ruleID` | `string` | ID of the matching rule to update. |
| `Name` | `string` | Updated name for the rule. |
| `Description` | `string` | Updated description of what the rule does. |
| `Criteria` | `[]blnkgo.`<br />`Criteria` | Full replacement array of match criteria. |
| `Criteria.`<br />`[].Field` | `blnkgo.`<br />`CriteriaField` | Field to match: `CriteriaField`<br />`Amount`, `CriteriaField`<br />`Currency`, `CriteriaField`<br />`Reference`, `CriteriaField`<br />`Description`, or `CriteriaField`<br />`Date`. |
| `Criteria.`<br />`[].Operator` | `blnkgo.`<br />`ReconciliationOperator` | How to compare the field: `ReconciliationOperator`<br />`Equals` or `ReconciliationOperator`<br />`Contains`. |
| `Criteria.`<br />`[].AllowableDrift` | `float64` | Optional tolerance for `amount` (percentage) or `date` (seconds). |

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "rule_id": "rule_890bdbc4-467f-4670-8424-5667e41daf29",
  "name": "Amount match (updated)",
  "description": "Match by amount with 2% drift",
  "criteria": [
    {
      "field": "amount",
      "operator": "equals",
      "allowable_drift": 0.02
    }
  ],
  "created_at": "2024-11-26T08:36:36.238244338Z",
  "updated_at": "2024-12-01T10:15:00.000000000Z"
}
```

#### Go response (`*blnkgo.RunReconResp`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `RuleID` | `string` | Unchanged ID of the rule. |
| `Name` | `string` | Updated rule name. |
| `Description` | `string` | Updated description of what the rule does. |
| `Criteria` | `[]blnkgo.`<br />`Criteria` | The updated match criteria stored for the rule. |
| `CreatedAt` | `string` | When the rule was created. |
| `UpdatedAt` | `string` | When the rule was last updated. |

</Step>
</Steps>
