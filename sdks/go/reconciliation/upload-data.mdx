Use `client.Reconciliation.Upload` to upload external statement data.

<Steps>
<Step title="Call the method">

```go client.Reconciliation.Upload wrap
file, err := os.Open("statement.csv")
if err != nil {
	// handle error
}
defer file.Close()

uploadResp, resp, err := client.Reconciliation.Upload("stripe", file, file.Name())
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `source` | `string` | Label describing where the data came from, for example `stripe`. |
| `file` | `io.Reader` | Open file or other readable source containing the external records. |
| `fileName` | `string` | Filename sent to Core, typically `file.Name()`. |

</Step>

<Step title="Response">

```json 201 Created wrap
{
  "upload_id": "upload_8c700d1b-09c0-4ef4-9ab1-a0decf3d0aa3",
  "record_count": 1000,
  "source": "Stripe"
}
```

#### Go response (`*blnkgo.ReconciliationUploadResp`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `UploadID` | `string` | Unique ID for the upload. Store it to start a batch reconciliation run. |
| `RecordCount` | `int` | Number of records found in the file. |
| `Source` | `string` | Label you passed for the data origin. |

</Step>
</Steps>
