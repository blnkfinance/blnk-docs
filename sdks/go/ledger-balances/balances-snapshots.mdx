Use `client.LedgerBalance.CreateSnapshot` to trigger [balance snapshot](/balances/balance-snapshots) processing on demand.

<Steps>
<Step title="Call the method">

```go client.LedgerBalance.CreateSnapshot wrap
result, resp, err := client.LedgerBalance.CreateSnapshot(blnkgo.CreateBalanceSnapshotRequest{
	BatchSize: 500,
})
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `BatchSize` | `int` | Optional. Server default is 1000. Omit or set to `0` to use the default. |

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "message": "Snapshotting in progress. should be completed shortly"
}
```

#### Go response (`*blnkgo.CreateBalanceSnapshotResponse`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `Message` | `string` | Status message when the snapshot job starts. |

</Step>
</Steps>
