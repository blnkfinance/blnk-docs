Use `client.BalanceMonitor.Delete` to remove a [balance monitor](/balances/balance-monitoring) by its ID.

<Steps>
<Step title="Call the method">

```go client.BalanceMonitor.Delete wrap
deleted, resp, err := client.BalanceMonitor.Delete(
	"mon_e0e77b0c-4985-472a-9bf5-76a48b0259b0",
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `monitorID` | `string` | Monitor ID to delete. Must be non-empty. |

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "message": "BalanceMonitor deleted successfully"
}
```

#### Go response (`*blnkgo.DeleteBalanceMonitorResponse`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `Message` | `string` | Confirmation that the monitor was deleted. |

</Step>
</Steps>
