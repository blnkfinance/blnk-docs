Use `client.Hooks.Delete` to permanently remove a [webhook](/webhooks/overview) by its ID.

<Steps>
<Step title="Call the method">

```go client.Hooks.Delete wrap
deleted, resp, err := client.Hooks.Delete(
	"hk_890bdbc4-467f-4670-8424-5667e41daf29",
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `hookID` | `string` | ID of the hook to delete. |

</Step>

<Step title="Confirm the outcome">

Check `deleted.Message` to confirm the hook was removed. Blnk will no longer call this endpoint on transactions.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "message": "hook deleted successfully"
}
```

#### Go response (`*blnkgo.DeleteHookResponse`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `Message` | `string` | Confirmation that the hook was deleted. |

</Step>
</Steps>
