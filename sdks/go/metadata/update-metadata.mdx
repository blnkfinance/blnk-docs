Use `client.Metadata.UpdateMetadata` to add or change [metadata](/metadata/overview) on an existing ledger, transaction, balance, or identity.

<Steps>
<Step title="Call the method">

```go client.Metadata.UpdateMetadata wrap
result, resp, err := client.Metadata.UpdateMetadata(
	"ldg_073f7ffe-9dfd-42ce",
	blnkgo.UpdateMetaDataRequest{
		MetaData: blnkgo.MetaData{
			"project_owner": "Acme LLC",
			"update_status": "Approved",
		},
	},
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `entityID` | `string` | ID of the entity to update (`ldg_…`, `txn_…`, `bln_…`, or `idt_…`). |
| `MetaData` | `blnkgo.MetaData` | Key-value pairs to add or update on the entity. |

</Step>

<Step title="Response">

```json 201 Created wrap
{
  "meta_data": {
    "project_owner": "Acme LLC",
    "update_status": "Approved"
  }
}
```

#### Go response (`*blnkgo.Metadata`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `MetaData` | `blnkgo.MetaData` | The full metadata object after the update. |

</Step>
</Steps>
