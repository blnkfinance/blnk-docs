Use `client.Ledger.Update` to edit the name of a specific ledger.

<Note>To update your ledger metadata, use [client.Metadata.Update](/reference/update-metadata) instead.</Note>

<Steps>
<Step title="Call the method">

```go client.Ledger.Update wrap
ledger, resp, err := client.Ledger.Update(
	"ldg_073f7ffe-9dfd-42ce-aa50-d1dca1788adc",
	blnkgo.UpdateLedgerRequest{
		Name: "Updated Customer Savings Ledger",
	},
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `string` | Ledger ID to update. |
| `Name` | `string` | New display name for the ledger. |

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "ledger_id": "ldg_073f7ffe-9dfd-42ce-aa50-d1dca1788adc",
  "name": "Updated Customer Savings Ledger",
  "created_at": "2024-02-20T05:28:03.558281542Z",
  "meta_data": {
    "project_owner": "Acme LLC",
    "description": "For all savings account created & managed by Acme."
  }
}
```

#### Go response (`*blnkgo.Ledger`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `LedgerID` | `string` | Unique ID for the ledger. Unchanged after the update. |
| `Name` | `string` | Updated name you passed in the request. |
| `CreatedAt` | `time.Time` | Date and time the ledger was created. |
| `MetaData` | `MetaData` | Existing metadata on the ledger. Not modified by this method. |

</Step>
</Steps>
