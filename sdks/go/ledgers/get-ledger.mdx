Use `client.Ledger.Get` to retrieve a [ledger](/ledgers/general-ledger) by its ledger ID.

<Steps>
<Step title="Call the method">

```go client.Ledger.Get
ledger, resp, err := client.Ledger.Get(
	"ldg_049495c6-356e-4ebc",
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `string` | Ledger ID to retrieve. |

</Step>

<Step title="Use the ledger details">

Read the returned `ledger` for the name, metadata, and creation time. Use these values to confirm you have the correct ledger before creating balances or transactions.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "ledger_id": "ldg_049495c6-356e-4ebc-a45e-60d1e1e16afb",
  "name": "Customer Checking Ledger",
  "created_at": "2024-11-26T08:23:57.509804Z",
  "meta_data": {
    "project_owner": "Blnk Tech"
  }
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `ledger_id` | `string` | Unique ID for the ledger. |
| `name` | `string` | Current display name. |
| `created_at` | `string` | Date and time the ledger was created. |
| `meta_data` | `object` | Metadata object, if one was set at creation. |

</Step>
</Steps>
