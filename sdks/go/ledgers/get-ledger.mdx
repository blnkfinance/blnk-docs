Use `client.Ledger.Get` to retrieve a [ledger](/ledgers/general-ledger) by its ledger ID.

<Steps>
<Step title="Call the method">

```go client.Ledger.Get wrap
ledger, resp, err := client.Ledger.Get(
	"ldg_049495c6-356e-4ebc-a45e-60d1e1e16afb",
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `string` | Ledger ID to retrieve. |

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "ledger_id": "ldg_049495c6-356e-4ebc-a45e-60d1e1e16afb",
  "name": "Customer Checking Ledger",
  "created_at": "2024-11-26T08:23:57.509804Z",
  "meta_data": {
    "project_owner": "Blnk Tech"
  }
}
```

#### Go response (`*blnkgo.Ledger`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `LedgerID` | `string` | Unique ID for the ledger. |
| `Name` | `string` | Current display name. |
| `CreatedAt` | `time.Time` | Date and time the ledger was created. |
| `MetaData` | `MetaData` | Metadata object, if one was set at creation. |

</Step>
</Steps>
