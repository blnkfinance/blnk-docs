Use `client.Ledger.List` to fetch every [ledger](/ledgers/general-ledger) on your instance. Core applies default pagination (`limit=10`, `offset=0`) when no query parameters are passed.

<Steps>
<Step title="Call the method">

```go client.Ledger.List wrap
ledgers, resp, err := client.Ledger.List()
```

This method takes no arguments. For structured queries, use `client.Ledger.Filter` instead.

</Step>

<Step title="Response">

```json 200 OK wrap
[
  {
    "ledger_id": "ldg_073f7ffe-9dfd-42ce-aa50-d1dca1788adc",
    "name": "Customer wallets",
    "created_at": "2024-02-20T05:28:03.558281542Z",
    "meta_data": {
      "environment": "production"
    }
  }
]
```

#### Go response (`[]blnkgo.Ledger`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `LedgerID` | `string` | Unique ID for the ledger. |
| `Name` | `string` | Display name of the ledger. |
| `CreatedAt` | `time.Time` | Date and time the ledger was created. |
| `MetaData` | `MetaData` | Metadata object, if one was set at creation. |

</Step>
</Steps>
