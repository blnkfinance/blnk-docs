Use `client.Transaction.BulkCommitInflight` to commit multiple inflight transactions.

<Steps>
<Step title="Call the method">

<Tabs>
<Tab title="Queued commit">

```go client.Transaction.BulkCommitInflight wrap
result, resp, err := client.Transaction.BulkCommitInflight(
 blnkgo.BulkCommitInflightRequest{
	Transactions: []blnkgo.BulkCommitInflightItem{
	 {
		TransactionID: "txn_c4e70eb8-e4d6-4e04-a2e2-92a43b96",
	 },
	 {
		TransactionID: "txn_6164573b-6cc8-45a4-ad2e-7b4ba6a6",
		Amount: 40,
	 },
	 {
		TransactionID: "txn_0b59f6e6-6c4a-4efa-915c-526f77ef",
		PreciseAmount: big.NewInt(125034),
	 },
	},
 },
)
```

</Tab>

<Tab title="Synchronous commit">

Pass `SkipQueue: true` to process each commit synchronously instead of queuing.

```go client.Transaction.BulkCommitInflight wrap
result, resp, err := client.Transaction.BulkCommitInflight(
 blnkgo.BulkCommitInflightRequest{
	SkipQueue: true,
	Transactions: []blnkgo.BulkCommitInflightItem{
	 {
		TransactionID: "txn_c4e70eb8-e4d6-4e04-a2e2-92a43b96",
	 },
	 {
		TransactionID: "txn_6164573b-6cc8-45a4-ad2e-7b4ba6a6",
		Amount: 40,
	 },
	},
 },
)
```

</Tab>
</Tabs>

| Field | Type | Description |
| :-- | :-- | :-- |
| `Transactions` | `[]blnkgo.`<br />`BulkCommitInflightItem` | Up to 100 inflight transactions to commit. |
| `Transactions.`<br />`[].TransactionID` | `string` | Inflight transaction ID to commit. |
| `Transactions.`<br />`[].Amount` | `float64` | Partial commit amount. Omit to commit the full remaining inflight amount. |
| `Transactions.`<br />`[].PreciseAmount` | `*big.Int` | Precision-applied partial commit amount. Takes precedence over `Amount`. |
| `SkipQueue` | `bool` | When `true`, process synchronously instead of queuing. |

</Step>

<Step title="Response">

```json 200 OK wrap expandable
{
  "succeeded": 2,
  "failed": 1,
  "results": [
    {
      "transaction_id": "txn_c4e70eb8-e4d6-4e04-a2e2-92a43b96",
      "status": "succeeded"
    },
    {
      "transaction_id": "txn_6164573b-6cc8-45a4-ad2e-7b4ba6a6",
      "status": "succeeded"
    },
    {
      "transaction_id": "txn_0b59f6e6-6c4a-4efa-915c-526f77ef",
      "status": "failed",
      "code": "INVALID_AMOUNT",
      "message": "cannot commit more than inflight amount"
    }
  ]
}
```

#### Go response (`*blnkgo.BulkCommitInflightResponse`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `Succeeded` | `int` | Number committed successfully. |
| `Failed` | `int` | Number that failed. |
| `Results` | `[]blnkgo.`<br />`BulkCommitInflightResult` | Per-transaction outcome in request order. |
| `Results.`<br />`[].TransactionID` | `string` | Transaction ID from the request. |
| `Results.`<br />`[].Status` | `string` | `succeeded` or `failed`. |
| `Results.`<br />`[].Code` | `string` | Failure code (for example `NOT_FOUND`, `ALREADY_VOIDED`, `INVALID_AMOUNT`). |
| `Results.`<br />`[].Message` | `string` | Human-readable failure message. |

</Step>
</Steps>
