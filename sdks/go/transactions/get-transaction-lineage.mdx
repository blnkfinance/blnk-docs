Use `client.Transaction.GetLineage` to inspect a transaction's fund breakdown.

<Steps>
<Step title="Call the method">

```go client.Transaction.GetLineage wrap
lineage, resp, err := client.Transaction.GetLineage(
	"txn_8d2ce2f0-0d75-4a91-9d43-2ad2c2e6b9ad",
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `transactionID` | `string` | Transaction ID with fund lineage enabled. |

</Step>

<Step title="Response">

```json 200 OK wrap expandable
{
  "transaction_id": "txn_8d2ce2f0-0d75-4a91-9d43-2ad2c2e6b9ad",
  "fund_allocation": [
    {
      "provider": "stripe",
      "amount": "2500"
    }
  ],
  "shadow_transactions": [
    {
      "transaction_id": "txn_shadow_123",
      "reference": "ref_002_release_stripe_0",
      "precise_amount": "2500",
      "currency": "USD",
      "status": "APPLIED"
    }
  ]
}
```

#### Go response (`*blnkgo.TransactionLineage`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `TransactionID` | `string` | Transaction you queried. |
| `FundAllocation` | `[]map[string]`<br />`interface{}` | How much of the spend came from each provider. Amounts are strings in minor units. May be empty when no tagged funds were used. |
| `ShadowTransactions` | `[]map[string]`<br />`interface{}` | Internal records Blnk creates to track each provider's share of the spend. May be empty. |

</Step>
</Steps>
