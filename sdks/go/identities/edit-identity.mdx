Use `client.Identity.Update` to change the profile details on an [identity](/identities/introduction).

<Steps>
<Step title="Call the method">

```go client.Identity.Update wrap
identity, resp, err := client.Identity.Update(
	"idt_3b63c8da-af29-4cc3",
	&blnkgo.Identity{
		IdentityType: blnkgo.Individual,
		FirstName: "Jane",
		LastName: "Smith",
	},
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `identityId` | `string` | Identity ID to update. |
| `IdentityType` | `blnkgo.IdentityType` | `Individual` or `Organization`. |
| Profile fields | various | Any field from [create](/sdks/go/identities/create-identity) you want to change. |

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "identity_id": "idt_3b63c8da-af29-4cc3-ad38-df17d87456e6",
  "identity_type": "individual",
  "first_name": "Jane",
  "last_name": "Smith",
  "email_address": "alice.smith@example.com",
  "created_at": "2024-11-26T08:36:36.238244338Z"
}
```

#### Go response (`*blnkgo.IdentityResponse`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `IdentityId` | `string` | Unique ID for the identity. |
| `IdentityType` | `blnkgo.IdentityType` | Type of the identity. |
| `FirstName` | `string` | Updated first name. |
| `LastName` | `string` | Updated last name. |
| `EmailAddress` | `string` | Email address on the identity. |
| `CreatedAt` | `string` | Date and time the identity was created. |

</Step>
</Steps>
