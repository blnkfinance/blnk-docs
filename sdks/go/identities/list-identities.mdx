Use `client.Identity.List` to fetch every [identity](/identities/introduction) on your instance.

<Steps>
<Step title="Call the method">

```go client.Identity.List
identities, resp, err := client.Identity.List()
```

This method takes no arguments.

</Step>

<Step title="Use the identity list">

Iterate over the returned `[]*blnkgo.IdentityResponse` to find identities by profile fields or metadata. Save identity IDs you need for balance linking or profile updates.

</Step>

<Step title="Response">

```json 200 OK wrap
[
  {
    "identity_id": "idt_3b63c8da-af29-4cc3-ad38-df17d87456e6",
    "identity_type": "individual",
    "first_name": "Alice",
    "last_name": "Smith",
    "email_address": "alice.smith@example.com",
    "created_at": "2024-11-26T08:36:36.238244338Z"
  }
]
```

| Field | Type | Description |
| :-- | :-- | :-- |
| Response slice | `[]*blnkgo.IdentityResponse` | Array of identity objects returned by Core. |

</Step>
</Steps>
