Use `client.Identity.Delete` to permanently remove an [identity](/identities/introduction) by its ID.

<Steps>
<Step title="Call the method">

```go client.Identity.Delete wrap
deleted, resp, err := client.Identity.Delete(
	"idt_3b63c8da-af29-4cc3-ad38-df17d87456e6",
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `identityID` | `string` | Identity ID to delete. Must be non-empty. |

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "message": "Identity deleted successfully"
}
```

#### Go response (`*blnkgo.DeleteIdentityResponse`)

| Field | Type | Description |
| :-- | :-- | :-- |
| `Message` | `string` | Confirmation that the identity was deleted. |

</Step>
</Steps>
