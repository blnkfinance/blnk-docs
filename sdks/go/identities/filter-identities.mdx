Use `client.Identity.Filter` to query [identities](/identities/introduction) with structured filters.

<Note>
The Go SDK exposes `Filter` on each service rather than a unified `Search.filter` helper. Use `client.Identity.Filter` for identities, and see [filter records](/sdks/go/search/search-db) for other collections.
</Note>

<Steps>
<Step title="Call the method">

```go client.Identity.Filter
result, resp, err := client.Identity.Filter(
	blnkgo.FilterParams{
		Filters: []blnkgo.Filter{
			{
				Field: "email_address",
				Operator: blnkgo.OpEqual,
				Value: "john.doe@example.com",
			},
			{
				Field: "category",
				Operator: blnkgo.OpEqual,
				Value: "customer",
			},
		},
		Limit: 20,
		Offset: 0,
		IncludeCount: true,
	},
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `Filters` | `[]blnkgo.Filter` | Array of filter conditions, each with a `Field`, `Operator`, and a `Value` or `Values`. |
| `SortBy` | `string` | Field to sort by. Defaults to `created_at`. |
| `SortOrder` | `string` | Sort direction, `asc` or `desc`. Defaults to `desc`. |
| `IncludeCount` | `bool` | Return `TotalCount` in the response. Defaults to `false`. |
| `Limit` | `int` | Maximum records to return. Defaults to 20, max 100. |
| `Offset` | `int` | Records to skip for pagination. Defaults to 0. |

</Step>

<Step title="Use the filtered results">

Read `result.Data` for matching identities. Use `result.TotalCount` with `Limit` and `Offset` to paginate when `IncludeCount` is `true`.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "data": [
    {
      "identity_id": "idt_3b63c8da-af29-4cc3-ad38-df17d87456e6",
      "identity_type": "individual",
      "first_name": "John",
      "last_name": "Doe",
      "email_address": "john.doe@example.com",
      "category": "customer",
      "created_at": "2024-11-26T08:36:36.238244338Z"
    }
  ],
  "total_count": 1
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `data` | `object[]` | Array of matching identity records. |
| `total_count` | `number` | Total matching records. Only present when `IncludeCount` is `true`. |

</Step>
</Steps>
