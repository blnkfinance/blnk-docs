Use `blnk.ledger_balances.update_identity` to link a balance to an [identity](/identities/link-balances).

<Steps>
<Step title="Call the method">

```python blnk.ledger_balances.update_identity wrap
response = blnk.ledger_balances.update_identity(
  "bln_5ce86029-3c2e-4e2a-aae2-7fb931ca4c4f",
  {
    "identity_id": "idt_3b63c8da-af29-4cc3-ad38-df17d87456e6",
  },
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `balanceId` | `str` | Balance ID to update. |
| `identity_id` | `str` | Identity ID to link to the balance. |

</Step>

<Step title="Confirm the link">

On success, Core returns a confirmation message. Fetch the balance with [get balance](/sdks/python/ledger-balances/get-balance) to verify `identity_id` is set.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "message": "Balance identity updated successfully"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `message` | `str` | Confirmation from Core. |

</Step>
</Steps>
