Use `blnk.ledger_balances.create_snapshot` to trigger [balance snapshot](/balances/balance-snapshots) processing on demand.

<Steps>
<Step title="Call the method">

```python blnk.ledger_balances.create_snapshot wrap
response = blnk.ledger_balances.create_snapshot({
  "batch_size": 500,
})
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `batch_size` | `number` | Optional. Server default is 1000. |

</Step>

<Step title="Wait for completion">

Snapshot jobs run asynchronously. Core returns immediately with a status message while processing continues in the background.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "message": "Snapshotting in progress. should be completed shortly"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `message` | `str` | Status message when the snapshot job starts. |

</Step>
</Steps>
