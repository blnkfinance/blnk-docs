Use `blnk.hooks.get` to fetch a single [webhook](/hooks/overview) by its ID.

<Steps>
<Step title="Call the method">

```python blnk.hooks.get wrap
response = blnk.hooks.get("hk_test_123")
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `str` | ID of the hook to retrieve. |

</Step>

<Step title="Use the hook details">

Read `response.data` for URL, type, active status, and last execution results. Use this to debug hook failures or confirm configuration.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "id": "hk_test_123",
  "name": "Pre-transaction validation",
  "url": "https://api.example.com/validate",
  "type": "PRE_TRANSACTION",
  "active": true,
  "timeout": 30,
  "retry_count": 3,
  "created_at": "2024-11-26T08:36:36.238244338Z",
  "last_run": "2024-11-26T09:12:00.000000000Z",
  "last_success": true
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `str` | Unique ID for the hook. |
| `type` | `str` | `PRE_TRANSACTION` or `POST_TRANSACTION`. |
| `active` | `bool` | Whether the hook is enabled. |
| `last_run` | `str` | Date and time the hook last executed. |
| `last_success` | `bool` | Whether the most recent execution succeeded. |

</Step>
</Steps>
