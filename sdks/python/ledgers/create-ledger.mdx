Use `blnk.ledgers.create` to create a new [ledger](/ledgers/general-ledger) for your system.

<Steps>
<Step title="Call the method">

```python blnk.ledgers.create wrap
response = blnk.ledgers.create({
  "name": "Customer wallets",
  "meta_data": {
    "environment": "production",
  },
})
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `name` | `str` | Display name for the ledger. |
| `meta_data` | `dict` | Optional metadata for extra context. |

</Step>

<Step title="Save the ledger ID">

Use `response.data["ledger_id"]` when creating balances or transactions in this ledger.

</Step>

<Step title="Response">

```json 201 Created wrap
{
  "ledger_id": "ldg_073f7ffe-9dfd-42ce-aa50-d1dca1788adc",
  "name": "Customer wallets",
  "created_at": "2024-02-20T05:28:03.558281542Z",
  "meta_data": {
    "environment": "production"
  }
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `ledger_id` | `str` | Unique ID for the ledger. Use this when creating balances and transactions. |
| `name` | `str` | Name you passed in the request. |
| `created_at` | `str` | Date and time the ledger was created. |
| `meta_data` | `dict` | Metadata object, if you sent one. |

</Step>
</Steps>
