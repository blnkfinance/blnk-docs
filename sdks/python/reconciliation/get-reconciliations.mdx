Use `blnk.reconciliation.get` to fetch the status and match counts for a [reconciliation](/reconciliations/overview) run.

<Steps>
<Step title="Call the method">

```python blnk.reconciliation.get wrap
response = blnk.reconciliation.get(
  "rec_a8f2c91d-1234-4abc-9def-0123456789ab",
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `reconciliationId` | `str` | ID of the reconciliation run to retrieve. |

</Step>

<Step title="Use the reconciliation details">

Read `response.data` for status, matched and unmatched counts, and creation time. Poll this endpoint after starting a batch or instant run until status is `completed`.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "reconciliation_id": "rec-123",
  "status": "completed",
  "matched_transactions": 20,
  "unmatched_transactions": 5,
  "created_at": "2025-03-17T12:00:00Z"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `reconciliation_id` | `str` | ID of the reconciliation run. |
| `status` | `str` | Current status of the run, for example `completed`. |
| `matched_transactions` | `number` | Number of records that matched. |
| `unmatched_transactions` | `number` | Number of records that did not match. |
| `created_at` | `str` | When the reconciliation started. |

</Step>
</Steps>
