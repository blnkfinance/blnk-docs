Use `blnk.reconciliation.upload` to upload external statement data.

<Steps>
<Step title="Call the method">

```python blnk.reconciliation.upload wrap
response = blnk.reconciliation.upload("statement.csv", "stripe")
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `fileInput` | `str` | Path to a CSV or JSON file, or a readable stream containing the external records. |
| `source` | `str` | Label describing where the data came from, for example `stripe`. |

</Step>

<Step title="Save the upload ID">

Use `response.data["upload_id"]` as the `upload_id` when you [start reconciliation](/sdks/python/reconciliation/start-reconciliation).

</Step>

<Step title="Response">

```json 201 Created wrap
{
  "upload_id": "upload_8c700d1b-09c0-4ef4-9ab1-a0decf3d0aa3",
  "record_count": 1000,
  "source": "Stripe"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `upload_id` | `str` | Unique ID for the upload. Store it to start a batch reconciliation run. |
| `record_count` | `number` | Number of records found in the file. |
| `source` | `str` | Label you passed for the data origin. |

</Step>
</Steps>
