Use `blnk.reconciliation.delete_matching_rule` to delete a matching rule.

<Steps>
<Step title="Call the method">

```python blnk.reconciliation.delete_matching_rule wrap
response = blnk.reconciliation.delete_matching_rule("rule_abc123")
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `ruleId` | `str` | ID of the matching rule to delete. |

</Step>

<Step title="Confirm the outcome">

Check the response message to confirm the rule was removed. Reconciliation runs that reference this rule ID will no longer apply it.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "message": "Matching rule deleted successfully"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `message` | `str` | Confirmation that the rule was deleted. |

</Step>
</Steps>
