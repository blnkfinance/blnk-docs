Use `blnk.api_keys.delete` to revoke an [API key](/advanced/secure-blnk) by ID.

<Steps>
<Step title="Call the method">

```python blnk.api_keys.delete wrap
response = blnk.api_keys.delete("api_key_abc123", {
  "owner": "merchant_a",
})
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `str` | ID of the API key to revoke. |
| `owner` | `str` | Owner of the key. Required when authenticating with the master key. |

</Step>

<Step title="Confirm the outcome">

On success, Core returns `204 No Content` and `response.data` is `None`. The key can no longer authenticate requests.

</Step>

<Step title="Response">

```json 204 No Content wrap
{}
```

Returns `204 No Content` on success with an empty body, so `response.data` is `None`.

</Step>
</Steps>
