Use `blnk.system.health` to check whether Core is reachable.

<Steps>
<Step title="Call the method">

```python blnk.system.health wrap
response = blnk.system.health()
```

This method takes no arguments.

</Step>

<Step title="Confirm Core is healthy">

Check that `response.data["status"]` is `UP` before running other SDK calls in health checks or startup probes.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "status": "UP"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `status` | `str` | `UP` when Core is healthy and reachable. |

</Step>
</Steps>
