Use `blnk.identity.update` to change the profile details on an [identity](/identities/introduction).

<Steps>
<Step title="Call the method">

```python blnk.identity.update wrap
response = blnk.identity.update(
  "idt_3b63c8da-af29-4cc3-ad38-df17d87456e6",
  {
    "identity_type": "individual",
    "first_name": "Jane",
    "last_name": "Smith",
  },
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `str` | Identity ID to update. |
| `identity_type` | `str` | `individual` or `organization`. |
| Profile fields | `string` | Any field from [create](/sdks/python/identities/create-identity) you want to change. |

</Step>

<Step title="Verify the changes">

Read `response.data` to confirm the updated profile fields. The identity ID stays the same.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "identity_id": "idt_3b63c8da-af29-4cc3-ad38-df17d87456e6",
  "identity_type": "individual",
  "first_name": "Jane",
  "last_name": "Smith",
  "email_address": "alice.smith@example.com",
  "created_at": "2024-11-26T08:36:36.238244338Z"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `identity_id` | `str` | Unique ID for the identity. |
| `created_at` | `str` | Date and time the identity was created. |

The response returns the full identity with your changes applied.

</Step>
</Steps>
