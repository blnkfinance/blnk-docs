Use `blnk.identity.detokenize_field` to detokenize one identity field.

<Steps>
<Step title="Call the method">

```python blnk.identity.detokenize_field wrap
response = blnk.identity.detokenize_field(
  "idt_3b63c8da-af29-4cc3-ad38-df17d87456e6",
  "EmailAddress",
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `str` | Identity ID. |
| `field` | `str` | PascalCase field name to detokenize. |

</Step>

<Step title="Use the plaintext value">

Read `response.data["value"]` for the original field value. Handle it securely and avoid logging it.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "field": "EmailAddress",
  "value": "alice.smith@example.com"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `field` | `str` | Field you detokenized. |
| `value` | `str` | Original plaintext value. |

</Step>
</Steps>
