Use `blnk.identity.create` to create an [identity](/identities/introduction) for an individual or organization.

<Steps>
<Step title="Call the method">

```python blnk.identity.create wrap
response = blnk.identity.create({
  "identity_type": "individual",
  "first_name": "Alice",
  "last_name": "Smith",
  "email_address": "alice.smith@example.com",
})
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `identity_type` | `str` | `individual` or `organization`. |

<Expandable title="Optional parameters">

| Field | Type | Description |
| :-- | :-- | :-- |
| `identity_id` | `str` | Optional ID you can set yourself. Blnk generates one if you omit it. |
| `organization_name` | `str` | Name of the organization. |
| `first_name` | `str` | First name of the individual. |
| `last_name` | `str` | Last name of the individual. |
| `other_names` | `str` | Any other names for the individual. |
| `gender` | `str` | Gender of the individual. |
| `dob` | `str` | Date of birth (ISO 8601, for example `1990-01-15T00:00:00Z`). |
| `email_address` | `str` | Email address. |
| `phone_number` | `str` | Contact number. |
| `nationality` | `str` | Nationality of the individual. |
| `category` | `str` | Your own customer category. |
| `street` | `str` | Street address. |
| `country` | `str` | Country of residence. |
| `state` | `str` | State of residence. |
| `post_code` | `str` | Postal code. |
| `city` | `str` | City of residence. |
| `meta_data` | `dict` | Optional metadata. |

</Expandable>

</Step>

<Step title="Save the identity ID">

Use `response.data["identity_id"]` to [link balances](/identities/link-balances) or update profile details later.

</Step>

<Step title="Response">

```json 201 Created wrap
{
  "identity_id": "idt_11111111-1111-4111-8111-111111111111",
  "identity_type": "individual",
  "first_name": "Alice",
  "last_name": "Smith",
  "email_address": "alice.smith@example.com",
  "phone_number": "+1234567890",
  "nationality": "Canadian",
  "category": "customer",
  "created_at": "2024-11-26T08:36:36.238244338Z",
  "meta_data": {
    "customer_id": "CUST123456"
  }
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `identity_id` | `str` | Unique ID for the identity. Use it to [link balances](/identities/link-balances). |
| `identity_type` | `str` | Type you passed in the request. |
| `created_at` | `str` | Date and time the identity was created. |
| `meta_data` | `dict` | Metadata stored with the identity. |

The response echoes every field you sent.

</Step>
</Steps>
