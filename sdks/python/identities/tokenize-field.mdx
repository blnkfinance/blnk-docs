Use `blnk.identity.tokenize_field` to tokenize one identity field.

<Steps>
<Step title="Call the method">

```python blnk.identity.tokenize_field wrap
response = blnk.identity.tokenize_field(
  "idt_3b63c8da-af29-4cc3-ad38-df17d87456e6",
  "EmailAddress",
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `str` | Identity ID. |
| `field` | `str` | PascalCase field name to tokenize. |

</Step>

<Step title="Confirm tokenization">

Check the response message, then use [get tokenized fields](/sdks/python/identities/get-tokenized-fields) to verify the field is protected.

</Step>

<Step title="Response">

```json 201 Created wrap
{
  "message": "Field tokenized successfully"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `message` | `str` | Confirmation that the field was tokenized. |

</Step>
</Steps>
