Use `blnk.identity.tokenize` to tokenize several identity fields.

<Steps>
<Step title="Call the method">

```python blnk.identity.tokenize wrap
response = blnk.identity.tokenize(
  "idt_3b63c8da-af29-4cc3-ad38-df17d87456e6",
  {
    "fields": ["FirstName", "EmailAddress"],
  },
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `str` | Identity ID. |
| `fields` | `string[]` | PascalCase field names to tokenize. At least one required. |

</Step>

<Step title="Confirm tokenization">

Check the response message, then use [get tokenized fields](/sdks/python/identities/get-tokenized-fields) to verify which fields are protected.

</Step>

<Step title="Response">

```json 201 Created wrap
{
  "message": "Fields tokenized successfully"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `message` | `str` | Confirmation that the fields were tokenized. |

</Step>
</Steps>
