Use `blnk.transactions.get_by_reference` to retrieve a transaction by reference.

<Steps>
<Step title="Call the method">

```python blnk.transactions.get_by_reference wrap
response = blnk.transactions.get_by_reference(
  "ref_04551509-d7d3-4eab-a1fd-2eb12809b5a4",
)
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `reference` | `str` | Unique reference passed when the transaction was created. |

</Step>

<Step title="Use the transaction details">

Use the returned transaction ID when you need to refund, update inflight status, or inspect lineage.

</Step>

| Field | Type | Description |
| :-- | :-- | :-- |
| `transaction_id` | `str` | Unique ID for the transaction. |
| `reference` | `str` | Reference you queried. |

</Step>
