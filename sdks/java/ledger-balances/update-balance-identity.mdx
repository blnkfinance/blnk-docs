Use `blnk.ledgerBalances().updateIdentity` for this operation.

<Steps>
<Step title="Call the method">

```java wrap
String balanceId = "bln_5ce86029-3c2e-4e2a-aae2-7fb931ca4c4f";
String identityId = "idt_073f7ffe-9dfd-42ce-aa50-d1dca1788adc";

ApiResponse<JsonNode> response = blnk.ledgerBalances().updateIdentity(
    balanceId,
    UpdateBalanceIdentity.create().identityId(identityId));
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `balanceId` | `String` | Balance ID to update. |
| `identity_id` | `String` | Identity ID to link to the balance. |

</Step>

<Step title="Confirm the link">

On success, Core returns a confirmation message. Fetch the balance with [get balance](/sdks/java/ledger-balances/get-balance) to verify `identity_id` is set.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "message": "Balance identity updated successfully"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `message` | `String` | Confirmation from Core. |

</Step>
</Steps>
