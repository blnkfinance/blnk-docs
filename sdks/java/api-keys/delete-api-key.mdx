Use `blnk.apiKeys().delete` for this operation.

<Steps>
<Step title="Call the method">

```java wrap
String apiKeyId = "api_key_073f7ffe-9dfd-42ce-aa50-d1dca1788adc";

ApiResponse<JsonNode> response = blnk.apiKeys().delete(
    apiKeyId,
    DeleteApiKeyOptions.create().owner("merchant_a"));
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `String` | ID of the API key to revoke. |
| `owner` | `String` | Owner of the key. Required when authenticating with the master key. |

</Step>

<Step title="Confirm the outcome">

On success, Core returns `204 No Content` and `response.data()` is `null`. The key can no longer authenticate requests.

</Step>

<Step title="Response">

```json 204 No Content wrap
{}
```

Returns `204 No Content` on success with an empty body, so `response.data()` is `null`.

</Step>
</Steps>
