Use `blnk.identity().tokenize` for this operation.

<Steps>
<Step title="Call the method">

```java wrap
String identityId = "idt_073f7ffe-9dfd-42ce-aa50-d1dca1788adc";

ApiResponse<JsonNode> response = blnk.identity().tokenize(
    identityId,
    TokenizeIdentityData.create().fields(
        "EmailAddress",
        "PhoneNumber"
    )
);
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `String` | Identity ID. |
| `fields` | `List<String>` | PascalCase field names to tokenize. At least one required. |

</Step>

<Step title="Confirm tokenization">

Check the response message, then use [get tokenized fields](/sdks/java/identities/get-tokenized-fields) to verify which fields are protected.

</Step>

<Step title="Response">

```json 201 Created wrap
{
  "message": "Fields tokenized successfully"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `message` | `String` | Confirmation that the fields were tokenized. |

</Step>
</Steps>
