Use `blnk.identity().tokenizeField` for this operation.

<Steps>
<Step title="Call the method">

```java wrap
String identityId = "idt_073f7ffe-9dfd-42ce-aa50-d1dca1788adc";

ApiResponse<JsonNode> response = blnk.identity().tokenizeField(identityId, "FirstName");
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `String` | Identity ID. |
| `field` | `String` | PascalCase field name to tokenize. |

</Step>

<Step title="Confirm tokenization">

Check the response message, then use [get tokenized fields](/sdks/java/identities/get-tokenized-fields) to verify the field is protected.

</Step>

<Step title="Response">

```json 201 Created wrap
{
  "message": "Field tokenized successfully"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `message` | `String` | Confirmation that the field was tokenized. |

</Step>
</Steps>
