Use `blnk.identity().detokenizeField` for this operation.

<Steps>
<Step title="Call the method">

```java wrap
String identityId = "idt_073f7ffe-9dfd-42ce-aa50-d1dca1788adc";

ApiResponse<JsonNode> response = blnk.identity().detokenizeField(identityId, "FirstName");
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `String` | Identity ID. |
| `field` | `String` | PascalCase field name to detokenize. |

</Step>

<Step title="Use the plaintext value">

Read `response.data().get("value")` for the original field value. Handle it securely and avoid logging it.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "field": "EmailAddress",
  "value": "alice.smith@example.com"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `field` | `String` | Field you detokenized. |
| `value` | `String` | Original plaintext value. |

</Step>
</Steps>
