Use `blnk.metadata().update` for this operation.

<Steps>
<Step title="Call the method">

```java wrap
String ledgerId = "ldg_073f7ffe-9dfd-42ce-aa50-d1dca1788adc";

ApiResponse<JsonNode> response = blnk.metadata().update(
    ledgerId,
    UpdateMetadataData.create()
        .metaData(Map.of("environment", "production")));
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `String` | ID of the entity to update (`ldg_…`, `txn_…`, `bln_…`, or `idt_…`). |
| `data.meta_data` | `Map<String, Object>` | Key-value pairs to add or update on the entity. |

</Step>

<Step title="Use the updated metadata">

Read `response.data().get("meta_data")` for the full metadata object after the update. New keys are merged with existing metadata.

</Step>

<Step title="Response">

```json 201 Created wrap
{
  "meta_data": {
    "project_owner": "Acme LLC",
    "update_status": "Approved"
  }
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `meta_data` | `JsonNode` | The full metadata object after the update. |

</Step>
</Steps>
