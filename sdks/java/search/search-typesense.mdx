Use `blnk.search().search` for this operation.

<Steps>
<Step title="Call the method">

```java wrap
ApiResponse<JsonNode> response = blnk.search().search(
    SearchParams.create()
        .q("jane@example.com")
        .queryBy("email_address")
        .perPage(10),
    "identities");
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `q` | `String` | Search query text. Use `*` to return all records. |
| `query_by` | `String` | Comma-separated list of fields to search in. |
| `filter_by` | `String` | Filter conditions to refine results. |
| `sort_by` | `String` | Sort order, for example `created_at:desc`. |
| `page` | `Number` | Page number, starting at 1. |
| `per_page` | `Number` | Results per page. Max 250, defaults to 10. |
| `collection` | `String` | Collection to search: `ledgers`, `transactions`, `balances`, or `identities`. |

</Step>

<Step title="Use the search results">

Read `response.data().get("hits")` for matching documents. Each hit includes a `document` and optional `highlights` and `text_match` score.

</Step>

| Field | Type | Description |

| `hits` | `JsonNode` | JSON array of search results. Each element includes a `document` and optional `highlights` and `text_match`. |

</Step>
