Use `blnk.system().health` for this operation.

<Steps>
<Step title="Call the method">

```java wrap
ApiResponse<JsonNode> response = blnk.system().health();
```

This method takes no arguments.

</Step>

<Step title="Confirm Core is healthy">

Check that `response.data().get("status").asText()` is `UP` before running other SDK calls in health checks or startup probes.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "status": "UP"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `status` | `String` | `UP` when Core is healthy and reachable. |

</Step>
</Steps>
