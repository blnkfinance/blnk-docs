Use `blnk.System.health` to check whether Core is reachable.

```typescript blnk.System.health wrap {1}
const response = await blnk.System.health();
```

This method takes no arguments.

---

## Response shape

```json 200 OK wrap
{
  "status": "UP"
}
```

| Field | Description |
| :-- | :-- |
| `status` | `UP` when Core is healthy and reachable. |
