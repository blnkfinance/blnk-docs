Run a minimal SDK request. You need a running [Blnk Core instance](/home/install) and the [SDK installed](/sdks/typescript) before continuing.

<Steps>
<Step title="Create a script">

Create `index.ts`:

```typescript index.ts wrap expandable
import { BlnkInit } from '@blnkfinance/blnk-typescript';

const blnk = BlnkInit('', { baseUrl: 'http://localhost:5001' });

const response = await blnk.System.health();

if (response.status !== 200 || response.data?.status !== 'UP') {
  throw new Error(response.message);
}
```
</Step>

<Step title="Run it">
Run the script with `npx tsx index.ts`.
</Step>
</Steps>
