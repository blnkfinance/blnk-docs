Use `blnk.Reconciliation.deleteMatchingRule` to delete a matching rule.

<Steps>
<Step title="Call the method">

```typescript blnk.Reconciliation.deleteMatchingRule wrap
const response = await blnk.Reconciliation.deleteMatchingRule('rule_abc123');
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `ruleId` | `string` | ID of the matching rule to delete. |

</Step>

<Step title="Confirm the outcome">

Check the response message to confirm the rule was removed. Reconciliation runs that reference this rule ID will no longer apply it.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "message": "Matching rule deleted successfully"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `message` | `string` | Confirmation that the rule was deleted. |

</Step>
</Steps>
