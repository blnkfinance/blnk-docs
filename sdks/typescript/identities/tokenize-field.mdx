Use `blnk.Identity.tokenizeField` to tokenize one identity field.

<Steps>
<Step title="Call the method">

```typescript blnk.Identity.tokenizeField wrap
const response = await blnk.Identity.tokenizeField(
  'idt_3b63c8da-af29-4cc3-ad38-df17d87456e6',
  'EmailAddress',
);
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `string` | Identity ID. |
| `field` | `string` | PascalCase field name to tokenize. |

</Step>

<Step title="Confirm tokenization">

Check the response message, then use [get tokenized fields](/sdks/typescript/identities/get-tokenized-fields) to verify the field is protected.

</Step>

<Step title="Response">

```json 201 Created wrap
{
  "message": "Field tokenized successfully"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `message` | `string` | Confirmation that the field was tokenized. |

</Step>
</Steps>
