Use `blnk.Identity.tokenizeField` to tokenize one identity field.

```typescript blnk.Identity.tokenizeField wrap {1}
const response = await blnk.Identity.tokenizeField(
  'idt_3b63c8da-af29-4cc3-ad38-df17d87456e6',
  'EmailAddress',
);
```

| Field | Description |
| :-- | :-- |
| `id` | Identity ID. |
| `field` | PascalCase field name to tokenize. |

---

## Response shape

```json 201 Created wrap
{
  "message": "Field tokenized successfully"
}
```

| Field | Description |
| :-- | :-- |
| `message` | Confirmation that the field was tokenized. |
