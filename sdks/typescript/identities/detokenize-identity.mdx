Use `blnk.Identity.detokenize` to detokenize several identity fields.

```typescript blnk.Identity.detokenize wrap {1}
const response = await blnk.Identity.detokenize(
  'idt_3b63c8da-af29-4cc3-ad38-df17d87456e6',
  { fields: ['FirstName', 'EmailAddress'] },
);
```

| Field | Description |
| :-- | :-- |
| `id` | Identity ID. |
| `fields` | PascalCase field names to detokenize. Pass `[]` for all tokenized fields. |

---

## Response shape

```json 200 OK wrap
{
  "fields": {
    "firstName": "Alice",
    "emailAddress": "alice.smith@example.com"
  }
}
```

| Field | Description |
| :-- | :-- |
| `fields` | Original values for the fields you requested. |
