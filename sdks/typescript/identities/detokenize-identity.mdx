Use `blnk.Identity.detokenize` to detokenize several identity fields.

<Steps>
<Step title="Call the method">

```typescript blnk.Identity.detokenize wrap
const response = await blnk.Identity.detokenize(
  'idt_3b63c8da-af29-4cc3-ad38-df17d87456e6',
  {
    fields: ['FirstName', 'EmailAddress'],
  },
);
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `string` | Identity ID. |
| `fields` | `string[]` | PascalCase field names to detokenize. Pass `[]` for all tokenized fields. |

</Step>

<Step title="Use the plaintext values">

Read `response.data.fields` for the original values. Handle these values securely and avoid logging them.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "fields": {
    "firstName": "Alice",
    "emailAddress": "alice.smith@example.com"
  }
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `fields` | `Record<string, string>` | Original values for the fields you requested. |

</Step>
</Steps>
