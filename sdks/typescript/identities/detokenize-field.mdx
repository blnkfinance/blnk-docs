Use `blnk.Identity.detokenizeField` to detokenize one identity field.

<Steps>
<Step title="Call the method">

```typescript blnk.Identity.detokenizeField wrap
const response = await blnk.Identity.detokenizeField(
  'idt_3b63c8da-af29-4cc3-ad38-df17d87456e6',
  'EmailAddress',
);
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `string` | Identity ID. |
| `field` | `string` | PascalCase field name to detokenize. |

</Step>

<Step title="Use the plaintext value">

Read `response.data.value` for the original field value. Handle it securely and avoid logging it.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "field": "EmailAddress",
  "value": "alice.smith@example.com"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `field` | `string` | Field you detokenized. |
| `value` | `string` | Original plaintext value. |

</Step>
</Steps>
