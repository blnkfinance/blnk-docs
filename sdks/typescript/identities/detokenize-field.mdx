Use `blnk.Identity.detokenizeField` to detokenize one identity field.

```typescript blnk.Identity.detokenizeField wrap {1}
const response = await blnk.Identity.detokenizeField(
  'idt_3b63c8da-af29-4cc3-ad38-df17d87456e6',
  'EmailAddress',
);
```

| Field | Description |
| :-- | :-- |
| `id` | Identity ID. |
| `field` | PascalCase field name to detokenize. |

---

## Response shape

```json 200 OK wrap
{
  "field": "EmailAddress",
  "value": "alice.smith@example.com"
}
```

| Field | Description |
| :-- | :-- |
| `field` | Field you detokenized. |
| `value` | Original plaintext value. |
