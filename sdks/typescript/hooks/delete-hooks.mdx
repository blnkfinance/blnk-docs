Use `blnk.Hooks.delete` to permanently remove a [webhook](/hooks/overview) by its ID.

<Steps>
<Step title="Call the method">

```typescript blnk.Hooks.delete wrap
const response = await blnk.Hooks.delete('hk_test_123');
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `string` | ID of the hook to delete. |

</Step>

<Step title="Confirm the outcome">

Check the response message to confirm the hook was removed. Blnk will no longer call this endpoint on transactions.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "message": "hook deleted successfully"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `message` | `string` | Confirmation that the hook was deleted. |

</Step>
</Steps>
