Use `blnk.Hooks.delete` to permanently remove a [webhook](/hooks/overview) by its ID.

```typescript blnk.Hooks.delete wrap {1}
const response = await blnk.Hooks.delete('hk_test_123');
```

| Field | Description |
| :-- | :-- |
| `id` | ID of the hook to delete. |

---

## Response shape

```json 200 OK wrap
{
  "message": "hook deleted successfully"
}
```

| Field | Description |
| :-- | :-- |
| `message` | Confirmation that the hook was deleted. |
