Use `blnk.ApiKeys.delete` to revoke an [API key](/advanced/secure-blnk) by ID.

<Steps>
<Step title="Call the method">

```typescript blnk.ApiKeys.delete wrap
const response = await blnk.ApiKeys.delete(
  'api_key_890bdbc4-467f-4670-8424-5667e41daf29',
  {
    owner: 'merchant_a',
  },
);
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `string` | ID of the API key to revoke. |
| `options.owner` | `string` | Owner of the key. Required when authenticating with the master key. |

</Step>

<Step title="Confirm the outcome">

On success, Core returns `204 No Content` and `response.data` is `null`. The key can no longer authenticate requests.

</Step>

<Step title="Response">

```json 204 No Content wrap
{}
```

Returns `204 No Content` on success with an empty body, so `response.data` is `null`.

</Step>
</Steps>
