Use `blnk.Ledgers.get` to retrieve a [ledger](/ledgers/general-ledger) by its `ledger_id`.

```typescript blnk.Ledgers.get wrap {1}
const response = await blnk.Ledgers.get(
  'ldg_049495c6-356e-4ebc-a45e-60d1e1e16afb',
);
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `string` | Ledger ID to retrieve. |

---

## Response shape

```json 200 OK wrap
{
  "ledger_id": "ldg_049495c6-356e-4ebc-a45e-60d1e1e16afb",
  "name": "Customer Checking Ledger",
  "created_at": "2024-11-26T08:23:57.509804Z",
  "meta_data": {
    "project_owner": "Blnk Tech"
  }
}
```

| Field | Description |
| :-- | :-- |
| `ledger_id` | Unique ID for the ledger. |
| `name` | Current display name. |
| `created_at` | Date and time the ledger was created. |
| `meta_data` | Metadata object, if one was set at creation. |
