Use `blnk.Metadata.update` to add or change [metadata](/metadata/overview) on an existing ledger, transaction, balance, or identity.

<Steps>
<Step title="Call the method">

```typescript blnk.Metadata.update wrap
const response = await blnk.Metadata.update(
  'ldg_073f7ffe-9dfd-42ce-aa50-d1dca1788adc',
  {
    meta_data: {
      project_owner: 'Acme LLC',
      update_status: 'Approved',
    },
  },
);
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `string` | ID of the entity to update (`ldg_…`, `txn_…`, `bln_…`, or `idt_…`). |
| `data.meta_data` | `Record<string, unknown>` | Key-value pairs to add or update on the entity. |

</Step>

<Step title="Use the updated metadata">

Read `response.data.meta_data` for the full metadata object after the update. New keys are merged with existing metadata.

</Step>

<Step title="Response">

```json 201 Created wrap
{
  "meta_data": {
    "project_owner": "Acme LLC",
    "update_status": "Approved"
  }
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `meta_data` | `Record<string, unknown>` | The full metadata object after the update. |

</Step>
</Steps>
