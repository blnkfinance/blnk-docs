Use `blnk.BalanceMonitor.update` to update a balance monitor.

```typescript blnk.BalanceMonitor.update wrap {1}
const response = await blnk.BalanceMonitor.update(
  'mon_e0e77b0c-4985-472a-9bf5-76a48b0259b0',
  {
    balance_id: 'bln_5ce86029-3c2e-4e2a-aae2-7fb931ca4c4f',
    condition: {
      field: 'credit_balance',
      operator: '<',
      value: 500,
      precision: 100,
    },
    call_back_url: 'https://api.example.com/balance-monitor',
  },
);
```

| Field | Description |
| :-- | :-- |
| `id` | Monitor ID to update. |
| `balance_id` | Balance to monitor. |
| `condition` | Threshold rule with `field`, `operator`, `value`, and `precision`. |
| `call_back_url` | Webhook URL when the condition is met. |
| `description` | Optional description for the monitor. |

---

## Response shape

```json 200 OK wrap
{
  "message": "BalanceMonitor updated successfully"
}
```

| Field | Description |
| :-- | :-- |
| `message` | Confirmation from Core. |
