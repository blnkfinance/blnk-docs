Use `blnk.LedgerBalances.createSnapshot` to trigger [balance snapshot](/balances/balance-snapshots) processing on demand.

```typescript blnk.LedgerBalances.createSnapshot wrap {1}
const response = await blnk.LedgerBalances.createSnapshot({ batch_size: 500 });
```

| Field | Description |
| :-- | :-- |
| `batch_size` | Optional. Server default is 1000. |

---

## Response shape

```json 200 OK wrap
{
  "message": "Snapshotting in progress. should be completed shortly"
}
```

| Field | Description |
| :-- | :-- |
| `message` | Status message when the snapshot job starts. |
