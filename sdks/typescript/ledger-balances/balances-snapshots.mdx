Use `blnk.LedgerBalances.createSnapshot` to trigger [balance snapshot](/balances/balance-snapshots) processing on demand.

<Steps>
<Step title="Call the method">

```typescript blnk.LedgerBalances.createSnapshot wrap
const response = await blnk.LedgerBalances.createSnapshot({
  batch_size: 500,
});
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `batch_size` | `number` | Optional. Server default is 1000. |

</Step>

<Step title="Wait for completion">

Snapshot jobs run asynchronously. Core returns immediately with a status message while processing continues in the background.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "message": "Snapshotting in progress. should be completed shortly"
}
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `message` | `string` | Status message when the snapshot job starts. |

</Step>
</Steps>
