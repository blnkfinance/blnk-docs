Use `blnk.Transactions.recoverQueue` to retry queued transactions.

```typescript blnk.Transactions.recoverQueue wrap {1}
const response = await blnk.Transactions.recoverQueue({ threshold: '5m' });
```

| Field | Description |
| :-- | :-- |
| `threshold` | Minimum age of transactions to recover (for example `5m`, `1h`). Default `2m`. Minimum 2 minutes. |

---

## Response shape

```json 200 OK wrap
{
  "recovered": 3,
  "threshold": "5m0s"
}
```

| Field | Description |
| :-- | :-- |
| `recovered` | Number of transactions re-enqueued for processing. |
| `threshold` | Threshold duration that was used. |
