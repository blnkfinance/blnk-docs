Use `blnk.Ledgers.get` to retrieve a list of all ledgers, or a single ledger by its `ledger_id`.

<Tabs>
<Tab title="Single ledger">

Pass a `ledger_id` to fetch one ledger.

```typescript blnk.Ledgers.get wrap {1}
const response = await blnk.Ledgers.get(
  'ldg_049495c6-356e-4ebc-a45e-60d1e1e16afb',
);
```

</Tab>
<Tab title="All ledgers">

Call without an `id` to list every ledger on your instance.

```typescript blnk.Ledgers.get wrap {1}
const response = await blnk.Ledgers.get();
```

</Tab>
</Tabs>

| Field | Type | Description |
| :-- | :-- | :-- |
| `id` | `string` | Optional. Ledger ID to retrieve one ledger. Omit to list all ledgers. |

---

## Response shape

For a single ledger, `response.data` is one object. When you omit `id`, `response.data` is an array and each element uses the same fields.

<CodeGroup>
```json Single ledger wrap
{
  "ledger_id": "ldg_049495c6-356e-4ebc-a45e-60d1e1e16afb",
  "name": "Customer Checking Ledger",
  "created_at": "2024-11-26T08:23:57.509804Z",
  "meta_data": {
    "project_owner": "Blnk Tech"
  }
}
```

```json All ledgers wrap
[
  {
    "ledger_id": "ldg_049495c6-356e-4ebc-a45e-60d1e1e16afb",
    "name": "Customer Checking Ledger",
    "created_at": "2024-11-26T08:23:57.509804Z",
    "meta_data": {
      "project_owner": "Blnk Tech"
    }
  },
  {
    "ledger_id": "ldg_845e6b1e-0463-4ce8-b858-2b3a398beff9",
    "name": "Customer Savings Ledger",
    "created_at": "2024-11-26T08:23:46.171319Z",
    "meta_data": {
      "project_owner": "Blnk Tech"
    }
  }
]
```
</CodeGroup>

| Field | Description |
| :-- | :-- |
| `ledger_id` | Unique ID for the ledger. |
| `name` | Current display name. |
| `created_at` | ISO 8601 timestamp from Core. |
| `meta_data` | Metadata object, if one was set at creation. |
