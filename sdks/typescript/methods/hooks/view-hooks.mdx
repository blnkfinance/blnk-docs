Use `blnk.Hooks.get` to fetch a single [webhook](/hooks/overview) by its ID.

```typescript blnk.Hooks.get wrap {1}
const response = await blnk.Hooks.get('hk_test_123');
```

| Field | Description |
| :-- | :-- |
| `id` | ID of the hook to retrieve. |

---

## Response shape

```json 200 OK wrap
{
  "id": "hk_test_123",
  "name": "Pre-transaction validation",
  "url": "https://api.example.com/validate",
  "type": "PRE_TRANSACTION",
  "active": true,
  "timeout": 30,
  "retry_count": 3,
  "created_at": "2024-11-26T08:36:36.238244338Z",
  "last_run": "2024-11-26T09:12:00.000000000Z",
  "last_success": true
}
```

| Field | Description |
| :-- | :-- |
| `id` | Unique ID for the hook. |
| `type` | `PRE_TRANSACTION` or `POST_TRANSACTION`. |
| `active` | Whether the hook is enabled. |
| `last_run` | Date and time the hook last executed. |
| `last_success` | Whether the most recent execution succeeded. |
