Use `blnk.ApiKeys.delete` to revoke an [API key](/advanced/secure-blnk) by ID.

```typescript blnk.ApiKeys.delete wrap {1}
const response = await blnk.ApiKeys.delete('api_key_abc123', {
  owner: 'merchant_a',
});
```

| Field | Description |
| :-- | :-- |
| `id` | ID of the API key to revoke. |
| `options.owner` | Owner of the key. Required when authenticating with the master key. |

---

## Response shape

```json 204 No Content wrap
{}
```

Returns `204 No Content` on success with an empty body, so `response.data` is `null`.
