Use `blnk.Reconciliation.deleteMatchingRule` to delete a matching rule.

```typescript blnk.Reconciliation.deleteMatchingRule wrap {1}
const response = await blnk.Reconciliation.deleteMatchingRule('rule_abc123');
```

| Field | Description |
| :-- | :-- |
| `ruleId` | ID of the matching rule to delete. |

---

## Response shape

```json 200 OK wrap
{
  "message": "Matching rule deleted successfully"
}
```

| Field | Description |
| :-- | :-- |
| `message` | Confirmation that the rule was deleted. |
