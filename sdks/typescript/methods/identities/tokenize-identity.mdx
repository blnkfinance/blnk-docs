Use `blnk.Identity.tokenize` to tokenize several identity fields.

```typescript blnk.Identity.tokenize wrap {1}
const response = await blnk.Identity.tokenize(
  'idt_3b63c8da-af29-4cc3-ad38-df17d87456e6',
  { fields: ['FirstName', 'EmailAddress'] },
);
```

| Field | Description |
| :-- | :-- |
| `id` | Identity ID. |
| `fields` | PascalCase field names to tokenize. At least one required. |

---

## Response shape

```json 201 Created wrap
{
  "message": "Fields tokenized successfully"
}
```

| Field | Description |
| :-- | :-- |
| `message` | Confirmation that the fields were tokenized. |
