Use `blnk.Search.getReindexStatus` to check search reindex progress.

<Steps>
<Step title="Call the method">

```typescript blnk.Search.getReindexStatus wrap
const response = await blnk.Search.getReindexStatus();

```

This method takes no arguments.

</Step>

<Step title="Track reindex progress">

Poll until `response.data.status` is `completed`. Compare `processed_records` to `total_records` to estimate remaining work.

</Step>

<Step title="Response">

```json 200 OK wrap
{
  "status": "in_progress",
  "phase": "indexing_transactions",
  "total_records": 2145709,
  "processed_records": 2145709,
  "started_at": "2026-02-12T12:13:10.448612096Z"
}

```

| Field | Type | Description |
| :-- | :-- | :-- |
| `status` | `string` | Current state, `in_progress` or `completed`. |
| `phase` | `string` | Current phase, for example `indexing_transactions`, or `done` when complete. |
| `total_records` | `number` | Total number of records to index. |
| `processed_records` | `number` | Number of records processed so far. |
| `started_at` | `string` | Timestamp when the reindex began. |
| `completed_at` | `string` | Timestamp when the reindex finished. Only present once `status` is `completed`. |
| `errors` | `object[]` | Array of error messages. Only present when an error occurs. |

</Step>
</Steps>
