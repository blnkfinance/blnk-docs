Use `blnk.Transactions.updateStatus` to commit or void an [inflight](/transactions/inflight) transaction or batch.

```typescript blnk.Transactions.updateStatus wrap {1}
const response = await blnk.Transactions.updateStatus(
  'txn_c4e70eb8-e4d6-4e04-a2e2-92a43b969e0c',
  { status: 'commit' },
);
```

| Field | Description |
| :-- | :-- |
| `transactionOrBatchId` | Inflight `transaction_id`, or `batch_id` from `createBulk`. |
| `status` | `commit` to apply funds, or `void` to cancel and roll back. |
| `amount` | Partial commit amount. See [Partial commits](/transactions/inflight#partial-commits). |
| `precise_amount` | Precision-applied partial commit amount. Takes precedence over `amount`. |

```json 200 OK wrap
{
  "amount": 1250.34,
  "rate": 0,
  "precision": 100,
  "precise_amount": 125034,
  "transaction_id": "txn_c4e70eb8-e4d6-4e04-a2e2-92a43b969e0c",
  "parent_transaction": "txn_c4e70eb8-e4d6-4e04-a2e2-92a43b969e0c",
  "source": "bln_f344b673-e855-4bda-b769-3e94a02c1941",
  "destination": "@WorldUSD",
  "reference": "ref_2ye281ewiu",
  "currency": "USD",
  "description": "Card payment on Stripe",
  "status": "COMMIT",
  "hash": "0b9c25fb5b00d6c71cb4ca87026bf6dc316e63353d3330deb588bd0b3d74dcc0",
  "created_at": "2024-11-26T09:33:35.265582042Z"
}
```

| Field | Description |
| :-- | :-- |
| `transaction_id` | ID of the new transaction created by the commit or void. |
| `parent_transaction` | Original inflight transaction this resolves. |
| `status` | `COMMIT` or `VOID`. |
| `precise_amount` | Amount applied (`amount` × `precision`). |
