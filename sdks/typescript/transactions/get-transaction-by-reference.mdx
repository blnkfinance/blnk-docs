Use `blnk.Transactions.getByReference` to retrieve a transaction by reference.

<Steps>
<Step title="Call the method">

```typescript blnk.Transactions.getByReference wrap
const response = await blnk.Transactions.getByReference(
  'ref_04551509-d7d3-4eab-a1fd-2eb12809b5a4',
);
```

| Field | Type | Description |
| :-- | :-- | :-- |
| `reference` | `string` | Unique reference passed when the transaction was created. |

</Step>

<Step title="Use the transaction details">

Use the returned transaction ID when you need to refund, update inflight status, or inspect lineage.

</Step>

<Step title="Response">

| Field | Type | Description |
| :-- | :-- | :-- |
| `transaction_id` | `string` | Unique ID for the transaction. |
| `reference` | `string` | Reference you queried. |
| `status` | `string` | Current [status](/transactions/transaction-lifecycle#statuses) of the transaction. |

</Step>
