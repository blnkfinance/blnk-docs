<Update label="1.1.0" description="Jun 18, 2026">
  ## Core 0.14.5 compatibility

  Targets Blnk Core `0.14.5`.

  ## Transactions

  Get transactions by ID or reference, read lineage, and bulk commit, void, or recover inflight batches.

  ## Ledgers and balances

  Update ledgers, query balances by indicator, link identities, capture snapshots, read historical balances, and manage balance monitors.

  ## Reconciliation and identity

  Upload external data, manage matching rules, run reconciliation, and tokenize or detokenize identity fields.

  ## Hooks, API keys, and system

  Manage transaction hooks, scoped API keys, metadata updates, and Core health checks.

  ## Fixes & improvements

  1. Configurable timeout, retries, base URL, and logger on `BlnkInit`.
  2. Consistent `ApiResponse` handling for legacy Core error bodies.
  3. Typesense search, DB filters, and reindex helpers on `blnk.Search`.
</Update>
