## Overview

Views let you save a set of filters so you can quickly return to them later. With views, you can switch between different data perspectives instantly, without having to recreate the same filters every time.

<img src="/cloud/img/filters/views.png" alt="Views bar showing saved custom views for quick access" className="rounded-lg" />

To create a new view:

1. Click the **Add filters** button in the top-left corner of the table or press **F** on your keyboard. 
2. Apply the filters you want to include in the view.
3. Click **Save as view** to save your current filter setup as a new view.
4. Enter a name for the view.
5. Add a short description to help you remember what the view is for (optional).
6. Click **Save view** to finish.

<img src="/cloud/img/filters/save-as-view-modal.png" alt="Save as view modal showing name and description fields" className="rounded-lg" />

## Edit views

To edit a view, navigate to the view on the views bar and click the **Edit filters** button. Edit your filters and click **Save changes** to update the view.

You can also rename the view or update its description before saving.

<img src="/cloud/img/filters/edit-view-modal.png" alt="Edit view modal showing options to modify filters and view details" className="rounded-lg" />

To delete a view, click on the **Delete view** button on the edit panel. 

<img src="/cloud/img/filters/delete-view-modal.png" alt="Delete view confirmation modal with warning message" className="rounded-lg" />
