A transaction stays inflight until it is acted upon or expired. You can complete it with `Commit` or cancel it with `Void`.

<img src="/cloud/img/transactions/inflight-actions.png" alt="Transaction action buttons including Commit and Actions menu" className="rounded-lg" />

## Commit transaction

<img src="/cloud/img/transactions/commit-transaction-modal.png" alt="Commit transaction modal with amount field" className="rounded-lg" />

To commit the full amount:

1. Select `Commit transaction` or press `C` on your keyboard.
2. Leave amount to commit empty.
3. Confirm in the modal.
4. A new linked transaction is created with status `APPLIED`.

For partial commits:

1. Enter the amount in `Amount to commit`.
2. A new linked transaction is created for that portion.
3. The remaining amount stays inflight.
4. For the final commit, leave `Amount to commit` empty.

## Void transaction

Voiding permanently cancels an inflight transaction and releases any remaining held amount.

<img src="/cloud/img/transactions/void-transaction.png" alt="Void transaction confirmation modal" className="rounded-lg" />

1. Select `Void transaction` or press `V` on your keyboard.
2. Confirm in the modal.
3. A new linked transaction is created with status `VOID`.
