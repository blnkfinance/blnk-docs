Use `Flag transaction` when a transaction needs manual review or investigation.

## Flag a transaction

<Steps>
  <Step title="Open transaction actions">
    Open the transaction details page and click `Actions`.

    <Frame>
      <img src="/cloud/img/transactions/flag-transaction-step-1-actions-menu.png" alt="Transaction details page with Actions button visible" className="rounded-lg" />
    </Frame>
  </Step>
  <Step title="Select Flag transaction">
    In the dropdown, select `Flag transaction`.

    <Frame>
      <img src="/cloud/img/transactions/flag-transaction-step-2-action-option.png" alt="Actions dropdown showing Flag transaction option" className="rounded-lg" />
    </Frame>
  </Step>
  <Step title="Complete the flag modal">
    In the modal, enter Title and Reason, then choose a Status.

    <Frame>
      <img src="/cloud/img/transactions/flag-transaction-step-3-modal.png" alt="Flag transaction modal with title, reason, and status fields" className="rounded-lg" />
    </Frame>
  </Step>
</Steps>

### Choose or create a status

Choose an existing status in the modal (for example, `Flagged`, `Investigating`, or `Resolved`). If you need a custom status, create one from Alert statuses and then return to the flag modal to select it.
