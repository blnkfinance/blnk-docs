Refunding sends funds back to the source balance by creating a linked refund record.

<Steps>
  <Step title="Open refund action">
    Open the transaction details page, then click `Refund transaction` or press `R`.

    <Frame>
      <img src="/cloud/img/transactions/refund-transaction-step-1-action.png" alt="Transaction details page with Refund transaction action" className="rounded-lg" />
    </Frame>
  </Step>
  <Step title="Confirm refund">
    In the confirmation modal, click `Refund transaction`.

    <Frame>
      <img src="/cloud/img/transactions/refund-transaction-step-2-modal.png" alt="Refund transaction confirmation modal" className="rounded-lg" />
    </Frame>
  </Step>
  <Step title="Verify result">
    A new linked refund record is created with status `APPLIED`.

    <Frame>
      <img src="/cloud/img/transactions/refund-transaction-step-3-result.png" alt="Transactions table showing the newly created refund record" className="rounded-lg" />
    </Frame>
  </Step>
</Steps>
