The General Ledger is a system-generated ledger that is automatically created for you when your instance is deployed.

It is best for grouping balances that belong to your organization, e.g., `@Revenue`, `@Fees`, etc.

Balances in the General Ledger are marked with the `@` icon indicator in your `Transactions` and `Balances` tables.

<img src="/cloud/img/ledgers/general-ledger-indicator.png" alt="General Ledger @ icon indicator in transactions and balances tables" className="rounded-lg" />

<Tip>For how the General Ledger works in Blnk, see the [general ledger guide](/ledgers/general-ledger).</Tip>

## Create general ledger balances

To create a balance in the General Ledger with the `@` indicator:

<Steps>
  <Step title="Create transaction">
    Go to `Transactions`. Click `Create transaction`, or press `âŚĄ + T` (`Alt + T` for Windows).

    <Frame>
      <img
        src="/cloud/img/ledgers/general-ledger-create-transaction-open.png"
        alt="Transactions table with Create transaction control open"
        className="rounded-lg"
      />
    </Frame>
  </Step>
  <Step title="Enter the required details">
    - `Amount`: The amount of the transaction.
    - `Currency`: The currency the transaction will use.
    - `Precision`: The multiplier that converts the amount into the currency's smallest unit.
    - `Description`: A short note describing the purpose of the transaction.

    Blnk Cloud automatically generates a reference number for the transaction.

    <Frame>
      <img
        src="/cloud/img/ledgers/general-ledger-create-transaction-amount.png"
        alt="Create transaction form with amount, currency, precision, and description filled in"
        className="rounded-lg"
      />
    </Frame>
  </Step>
  <Step title="Enter an indicator">
    In `Source` or `Destination`, enter an indicator instead of a balance ID. For example, enter `@Revenue` in `Destination` to create a revenue balance.

    <Frame>
      <img
        src="/cloud/img/ledgers/create-transaction-with-indicator.png"
        alt="Create transaction form with @Revenue in the Destination field"
        className="rounded-lg"
      />
    </Frame>
  </Step>
  <Step title="Complete">
    Click `Create transaction` to complete.

    <Frame>
      <img
        src="/cloud/img/ledgers/general-ledger-create-transaction-result.png"
        alt="Balances table showing a General Ledger balance with the @ indicator"
        className="rounded-lg"
      />
    </Frame>
  </Step>
</Steps>

Blnk checks if a balance exists with the given indicator. If it does, it creates a transaction to the balance.

If it doesn't, the balance is automatically created in your General Ledger with the given indicator and the transaction is mapped to it.

<Tip>Suggested values for a new `@Revenue` balance: Amount `500.00`, Currency `USD`, Precision `100`, Source `@FundingPool`, Destination `@Revenue`, Description `Platform revenue - May`. Enable `Apply overdraft` on the source if the balance has insufficient funds.</Tip>
