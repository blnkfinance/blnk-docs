The Ledger details page gives you a complete view of a ledger — including its summary information, linked balances, and all related activity.

<Frame>
  <img
    src="/cloud/img/ledgers/ledger-details-page.png"
    alt="Ledger details page showing summary and balances section"
    className="rounded-lg"
  />
</Frame>

## Summary section

At the top, you'll see your ledger information:


| Details | Description |
| :-- | :-- |
| **Name** | The name of the ledger |
| **Balances** | Number of balances currently in this ledger |
| **Transactions** | Total number of transactions involving this ledger |
| **Date created** | When the ledger was created |
| **Ledger ID** | Unique identifier for the ledger |

For more details, select **Full info** in the top-right corner of the summary.

## Create a balance in a ledger

To create a balance in the ledger you are viewing:

<Steps titleSize="h3">
  <Step title="Create balance">
    Click **Create balance** at the top right of the details page.

    <Frame>
      <img
        src="/cloud/img/ledgers/ledger-create-balance-open.png"
        alt="Ledger details page with Create balance control"
        className="rounded-lg"
      />
    </Frame>
  </Step>
  <Step title="Enter the currency">
    Enter the currency for the balance:

    - Use the ISO 4217 code for fiat currencies.
    - Use the ticker symbol for crypto.
    - Use a unique identifier for custom assets.

    <Frame>
      <img
        src="/cloud/img/ledgers/create-balance-in-ledger.png"
        alt="Create balance form within a ledger"
        className="rounded-lg"
      />
    </Frame>
  </Step>
  <Step title="Assign an owner (optional)">
    To link the balance to an identity, enable **Assign an owner to this balance**, then search by name and select from the list, or paste an identity ID.

    <Frame>
      <img
        src="/cloud/img/ledgers/ledger-create-balance-assign-owner.png"
        alt="Create balance form with Assign an owner to this balance enabled"
        className="rounded-lg"
      />
    </Frame>
  </Step>
  <Step title="Enable lineage tracking (optional)">
    To track where funds came from, enable **Enable lineage tracking**. Requires an owner on the balance.

    <Frame>
      <img
        src="/cloud/img/ledgers/ledger-create-balance-lineage.png"
        alt="Create balance form with Enable lineage tracking enabled"
        className="rounded-lg"
      />
    </Frame>
  </Step>
  <Step title="Complete">
    Click **Create balance** to finish.

    <Frame>
      <img
        src="/cloud/img/ledgers/ledger-create-balance-result.png"
        alt="Ledger details page showing a newly created balance"
        className="rounded-lg"
      />
    </Frame>
  </Step>
</Steps>

## Metadata

The metadata section lets you add custom information to your ledger.

<Steps titleSize="h3">
  <Step title="View metadata">
    Navigate to the **Extra information** section on the ledger details page.

    <Frame>
      <img
        src="/cloud/img/ledgers/ledger-metadata-section.png"
        alt="Extra information section on the ledger details page"
        className="rounded-lg"
      />
    </Frame>
  </Step>
  <Step title="Add or edit metadata">
    Click **Add** or **Edit** at the top right of the **Extra information** section.

    <Frame>
      <img
        src="/cloud/img/metadata/edit-metadata-button.png"
        alt="Add or Edit control in the Extra information section"
        className="rounded-lg"
      />
    </Frame>
  </Step>
  <Step title="Use a template (optional)">
    In the editor, click **Use template** to import keys from a [metadata template](/cloud/organization/metadata-templates), then fill in the values. You can also add keys with **+ Add metadata**.

    <Frame>
      <img
        src="/cloud/img/ledgers/ledger-metadata-use-template.png"
        alt="Metadata editor with Use template and Add metadata"
        className="rounded-lg"
      />
    </Frame>
  </Step>
  <Step title="Update">
    Click **Update** to save your changes.

    <Frame>
      <img
        src="/cloud/img/metadata/metadata-editor.png"
        alt="Metadata editor with ledger metadata filled in"
        className="rounded-lg"
      />
    </Frame>
  </Step>
</Steps>

<Note>When updating metadata, any fields you don't include in your update will remain unchanged. Blnk adds your new values to the existing metadata instead of deleting them. To remove a value, set it to an empty string.</Note>
