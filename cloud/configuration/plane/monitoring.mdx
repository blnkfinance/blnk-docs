Configure error tracking and server-side event tracking for Plane.

---

## Error and event tracking

```bash enterprise.env
SENTRY_DSN=
POSTHOG_API_KEY=
```

| Variable | Description | Default | Required |
| :-- | :-- | :-- | :-- |
| `SENTRY_DSN` | Sentry DSN for error reporting, tracing, and log capture. Plane initializes Sentry when `ENVIRONMENT=production`. | None | No |
| `POSTHOG_API_KEY` | PostHog project API key used by Plane server-side deployment event tracking. | None | No |

Leave both unset to avoid sending Plane telemetry to external monitoring services. Sentry initialization is gated by `ENVIRONMENT=production`.
