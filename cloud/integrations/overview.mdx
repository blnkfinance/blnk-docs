Integrations are how you connect Blnk Cloud to software outside the dashboard: your own services, automation scripts, AI assistants, and partner applications.

Use this section to choose the right authentication method and API surface, then follow the linked guides for setup and examples.

---

## How integrations work

Your client authenticates with Blnk Cloud. Cloud validates the request, routes it to the correct Core instance using `instance_id`, and returns the response. 

AI tools using MCP follow the same path: your assistant talks to Cloud, not directly to Core.

<img src="/cloud/img/auth/cloud-api.png" alt="Diagram showing a client authenticating with Blnk Cloud, which forwards requests to a Blnk Core instance" className="rounded-lg" />

<Note>
If you self-host Blnk Cloud under a production license, use your deployment's public URL as the API base instead of `https://api.cloud.blnkfinance.com`. See [License configuration](/cloud/license/configuration).
</Note>
