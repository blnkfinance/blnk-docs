Flag balance lets you raise an alert directly from a balance when you notice unusual activity or need a review trail.

When a balance is flagged, Blnk creates a linked alert so your team can assign ownership, update status, and resolve the issue from Alerts.

This page covers how to initiate flagging from a balance. For alert handling and resolution workflows, see [Alerts overview](/cloud/alerts).

---

## Flag a balance

To flag a balance from Cloud:

<Steps>
  <Step title="Open the balance actions">
    Open `Balances` and select the balance you want to review. In the details page, click `Actions`, then select `Flag balance`.

    <Frame>
      <img src="/cloud/img/balances/flag-balance-step-1-actions-menu.png" alt="Balance details page with Actions menu showing Flag balance" className="rounded-lg" />
    </Frame>
  </Step>
  <Step title="Enter the flag details">
    In the modal, enter Title and Reason, then choose a Status.

    <Frame>
      <img src="/cloud/img/balances/flag-balance-step-2-modal-fields.png" alt="Flag balance modal with title, reason, and status fields" className="rounded-lg" />
    </Frame>
  </Step>
  <Step title="Submit the flag">
    Click `Flag balance` to create the alert for this balance.

    <Frame>
      <img src="/cloud/img/balances/flag-balance-step-3-submitted.png" alt="Balance details page after a flag is created" className="rounded-lg" />
    </Frame>
  </Step>
</Steps>

<Tip>You can create multiple alerts on the same balance when different issues need separate tracking.</Tip>

### Choose or create a status

Choose an existing status in the modal (for example, `Flagged`, `Investigating`, or `Resolved`). If you need a custom status, create one from Alert statuses and then return to the flag modal to select it.
