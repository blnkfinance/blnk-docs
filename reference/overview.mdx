This section documents the **Blnk Core REST API.** It contains the endpoints your application calls to integrate with your running Blnk Core instance.

<Note>
**Important to note:**

If you're using the auto-provisioned `Enterprise Core` instance included with your Production License, set the base URL to: `https://ENTERPRISE_PUBLIC_URL/core`. 

Otherwise, use the publicly accessible base URL for your Core instance.
</Note>

Each page lists the HTTP method, path, parameters, example requests, and response fields for one operation. To successfully call an endpoint, authenticate requests with your API key in the `X-blnk-key` header if set to secure mode. 

If you are new to Blnk, start from the [documentation home](/), follow [installation](/home/install), then return here when you are ready to integrate.
