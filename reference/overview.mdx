This section documents the **Blnk Core REST API.** It contains the endpoints your application calls to integrate with your running Blnk Core instance.

Each page lists the HTTP method, path, parameters, example requests, and response fields for one operation.

Authenticate requests with your API key in the `X-blnk-key` header. 

If you are new to Blnk, start with [installation](/home/install), then return here when you are ready to integrate.
