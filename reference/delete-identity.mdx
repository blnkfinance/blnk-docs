### Authorization
Basic authentication header of the form — `Basic <encoded-value>`, where `<encoded-value>` is the base64-encoded string of your `username:password`.

### Path
<ParamField path="identity_id" type="string" required>
  Specifies the unique ID of the identity that needs to be deleted.
</ParamField>

<RequestExample>
```json
curl --request DELETE \
  --url http://localhost:5001/identities/{identity_id} \
  --header 'Authorization: Basic <encoded-value>' \
```
</RequestExample>
