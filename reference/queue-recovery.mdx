### Query parameters

<ParamField query="threshold" type="string">
  Minimum age of transactions to recover. Use duration format (e.g. `5m`, `10m`, `1h`). Default: `2m`. Minimum: 2 minutes.
</ParamField>

### Response

<ResponseField name="recovered" type="integer">
  Number of transactions re-enqueued for processing.
</ResponseField>

<ResponseField name="threshold" type="string">
  The threshold duration that was used (e.g. `2m0s`).
</ResponseField>

<RequestExample>
```bash
curl --request POST \
  --url 'http://localhost:5001/transactions/recover?threshold=5m' \
  --header 'X-blnk-key: <api-key>'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "recovered": 3,
  "threshold": "5m0s"
}
```
</ResponseExample>
