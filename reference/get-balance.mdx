### Path
<ParamField path="balance_id" type="string">
  Specifies the unique ID of the balance being retrieved. Leave empty to view all balances in your Blnk Ledger.
</ParamField>
<ParamField query="with_queued" type="boolean" default="false">
  Indicates whether to include the queued balance in the response.

  ```
  /balances/{balance_id}?with_queued=true
  ```
</ParamField>

{/* <Tip>
When listing all balances, you can use [advanced filter operators](/search/db/filtering) to narrow down results by currency, ledger, identity, balance amounts, and more.
</Tip> */}

<RequestExample>
```json
curl --request GET \
  --url http://localhost:5001/balances/{balance_id} \

```
