<Key />

### Path
<ParamField path="id" type="string" required>
  Unique `balance_id` of the balance whose identity you want to update.
</ParamField>

### Body
<ParamField body="identity_id" type="string" required>
  The new identity ID to associate with the balance. Must be a valid existing identity ID.
</ParamField>

<RequestExample>
```json
curl -X PUT "http://localhost:5001/balances/{id}/identity" \
  -H "X-blnk-key: <api-key>" \
  -H "Content-Type: application/json" \
  -d '{
    "identity_id": "idt_3b63c8da-af29-4cc3-ad38-df17d87456e6"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "message": "Balance identity updated successfully"
}
```

```json 404
{
  "error": "Identity with ID 'idt_00000000-0000-0000-0000-000000000000' not found",
  "error_detail": {
    "code": "BAL_NOT_FOUND",
    "message": "Identity with ID 'idt_00000000-0000-0000-0000-000000000000' not found"
  }
}
```
</ResponseExample>
