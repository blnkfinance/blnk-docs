### Path
<ParamField path="id" type="string" required>
  Unique `balance_id` of the balance whose identity you want to update.
</ParamField>

### Body
<ParamField body="identity_id" type="string" required>
  The new identity ID to associate with the balance. Must be a valid existing identity ID.
</ParamField>

<RequestExample>
```json
curl --request PUT \
  --url http://localhost:5001/balances/{id}/identity \
  --header 'X-blnk-key: <api-key>' \
  --header 'Content-Type: application/json' \
  --data '{
    "identity_id": "idt_3b63c8da-af29-4cc3-ad38-df17d87456e6"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "message": "Balance identity updated successfully"
}
```
