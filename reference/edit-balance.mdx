### Authorization
Basic authentication header of the form — `Basic <encoded-value>`, where `<encoded-value>` is the base64-encoded string of your `username:password`.

### Path
<ParamField path="balance_id" type="string">
  Unique `balance_id` of the balance to be update. 
</ParamField>

### Body
<ParamField body="ledger_id" type="string" required>
  Unique id of the ledger the balance is created in.
</ParamField>
<ParamField body="currency" type="string" required>
  The currency/asset class of this balance.
</ParamField>
<ParamField body="identity_id" type="string">
  Unique id of identity (if any) to be linked to the balance. Can be left out from the request body if there's no identity to be linked.
</ParamField>
<ParamField body="meta_data" type="object">
  Additional metadata for the request body. 
</ParamField>

<RequestExample>
```json
curl --request PUT \
  --url http://localhost:5001/balances/{balance_id} \
  --header 'Authorization: Basic <encoded-value>' \
  --header 'Content-Type: application/json' \
  --data '{
    "ledger_id": "ldg_049495c6-356e-4ebc-a45e-60d1e1e16afb",

  }'
```
