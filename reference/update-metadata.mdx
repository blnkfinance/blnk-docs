### Path
<ParamField path="id" type="string" required>
  The unique identifier of the ledger, transaction, identity, or balance whose metadata you want to update.
</ParamField>

### Body
<ParamField body="meta_data" type="object">
  Metadata information to be added or updated.
</ParamField>

<RequestExample>
```json
curl --request POST \
  --url http://localhost:5001/{id}/metadata \
  --header 'X-blnk-key: <api-key>' \
  --header 'Content-Type: application/json' \
  --data '{
    "meta_data": {
      "project_owner": "Acme LLC",
      "update_status": "Approved"
    }
  }'
```
</RequestExample>

<ResponseExample>
```json 201
{
  "meta_data": {
    "project_owner": "Acme LLC",
    "description": "Approved"
  }
}
```
</ResponseExample>
