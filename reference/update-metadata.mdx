### Path
<ParamField path="id" type="string" required>
  The unique identifier of the ledger, transaction, identity, or balance whose metadata you want to update.
</ParamField>

### Body
<ParamField body="meta_data" type="object">
  Metadata information to be added or updated.
</ParamField>

<RequestExample>
```json
curl -X POST "http://localhost:5001/{id}/metadata" \
  -H "X-blnk-key: <api-key>" \
  -H "Content-Type: application/json" \
  -d '{
    "meta_data": {
      "project_owner": "Acme LLC",
      "update_status": "Approved"
    }
  }'
```
</RequestExample>

<ResponseExample>
```json 201
{
  "meta_data": {
    "project_owner": "Acme LLC",
    "description": "Approved"
  }
}
```

```json 404
{
  "error": "entity not found",
  "error_detail": {
    "code": "META_ENTITY_NOT_FOUND",
    "message": "entity not found"
  }
}
```

```json 400
{
  "error": "unsupported entity type: zone",
  "error_detail": {
    "code": "META_UNSUPPORTED_ENTITY",
    "message": "unsupported entity type: zone"
  }
}
```
</ResponseExample>
