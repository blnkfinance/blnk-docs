### Path
<ParamField path="identity_id" type="string">
  Specifies the unique ID of the identity being retrieved
</ParamField>
<ParamField path="field" type="string">
  Specifies the field to be detokenized
</ParamField>

<ResponseExample>
```json 201
{
  "field": "EmailAddress",
  "value": "alice.smith@example.com"
}
```

```json 400
{
  "error": "field first_name is not tokenized",
  "error_detail": {
    "code": "IDT_FIELD_NOT_TOKENIZED",
    "message": "field first_name is not tokenized"
  }
}
```
