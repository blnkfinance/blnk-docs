### Path
<ParamField path="identity_id" type="string">
  Specifies the unique ID of the identity being retrieved. Leave empty to view all identities in your Blnk Ledger.
</ParamField>

{/* <Tip>
When listing all identities, you can use [advanced filter operators](/search/db/filtering) to narrow down results by name, email, category, country, and more.
</Tip> */}

<RequestExample>
```json
curl --request GET \
  --url http://localhost:5001/identities/{identity_id} \
  --header 'X-blnk-key: <api-key>'
```
</RequestExample>
