### Path
<ParamField path="identity_id" type="string">
  Specifies the unique ID of the identity being retrieved
</ParamField>

### Body
<ParamField body="fields" type="string[]">
  Specifies an array of fields to be tokenized.
</ParamField>

<RequestExample>
```bash
curl --request POST \
  --url http://localhost:5001/identities/{identity_id}/tokenize \
  --header 'X-blnk-key: <api-key>' \
  --header 'Content-Type: application/json' \
  --data '{
  "fields": [
    "FirstName", 
    "LastName", 
    "EmailAddress", 
    "PhoneNumber"
  ]
}'
```
</RequestExample>
<ResponseExample>
```json 201
{
    "message": "Fields tokenized successfully"
}
```
</ResponseExample>
