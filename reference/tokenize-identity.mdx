### Path
<ParamField path="identity_id" type="string">
  Specifies the unique ID of the identity being retrieved
</ParamField>

### Body
<ParamField body="fields" type="string[]">
  Specifies an array of fields to be tokenized.
</ParamField>

<RequestExample>
```bash
curl -X POST "http://localhost:5001/identities/{identity_id}/tokenize" \
  -H "X-blnk-key: <api-key>" \
  -H "Content-Type: application/json" \
  -d '{
  "fields": [
    "FirstName",
    "LastName",
    "EmailAddress",
    "PhoneNumber"
  ]
}'
```
</RequestExample>
<ResponseExample>
```json 201
{
    "message": "Fields tokenized successfully"
}
```

```json 400
{
  "error": "field first_name is not tokenizable",
  "error_detail": {
    "code": "IDT_FIELD_NOT_TOKENIZABLE",
    "message": "field first_name is not tokenizable"
  }
}
```
</ResponseExample>
