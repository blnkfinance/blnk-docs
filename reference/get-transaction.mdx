### Path
<ParamField path="transaction_id" type="string">
  Specifies the unique ID of the transaction record being retrieved. Leave empty to view all transactions in your Blnk Ledger.
</ParamField>

{/* <Tip>
When listing all transactions, you can use [advanced filter operators](/search/db/filtering) to narrow down results by status, currency, amount, source, destination, and more.
</Tip> */}

<RequestExample>
```json
curl --request GET \
  --url http://localhost:5001/transactions/{transaction_id} \
  --header 'X-blnk-key: <api-key>'
```
</RequestExample>
