### Body
<ParamField body="ledger_id" type="string" required>
  Unique id of the ledger the balance is created in.
</ParamField>
<ParamField body="currency" type="string" required>
  The currency/asset class of this balance. See the following: [Understanding asset classes](/balances/asset-classes) and [Currencies](/guides/currencies)
</ParamField>
<ParamField body="identity_id" type="string">
  Unique id of identity (if any) to be linked to the balance. Can be left out from the request body if there's no identity to be linked.
</ParamField>
<ParamField body="meta_data" type="object">
  Additional metadata for the request body. 
</ParamField>

### Response 
<ResponseField name="balance" type="integer" required>
  The net amount being stored in the balance.
</ResponseField>
<ResponseField name="inflight_balance" type="integer" required>
  The net amount waiting to be moved into or out of the balance.
</ResponseField>
<ResponseField name="credit_balance" type="integer" required>
  Total amount moved into the balance
</ResponseField>
<ResponseField name="inflight_credit_balance" type="integer" required>
  Total amount waiting to be moved into the balance.
</ResponseField>
<ResponseField name="debit_balance" type="integer" required>
  Total amount moved out of the balance.
</ResponseField>
<ResponseField name="inflight_debit_balance" type="integer" required>
  Total amount waiting to be moved out of the balance.
</ResponseField>
<ResponseField name="ledger_id" type="string" required>
  Unique id of the ledger the balance is created in.
</ResponseField>
<ResponseField name="identity_id" type="string">
  Unique id of the identity (if any) linked to this balance.
</ResponseField>
<ResponseField name="balance_id" type="string" required>
  Unique id of the created balance.
</ResponseField>
<ResponseField name="currency" type="string" required>
  The currency/asset class of the balance.
</ResponseField>
<ResponseField name="created_at" type="string" required>
  Indicates the date & time of creation.
</ResponseField>
<ResponseField name="meta_data" type="User Object">
  Additional metadata of the balance.
</ResponseField>

  }'
