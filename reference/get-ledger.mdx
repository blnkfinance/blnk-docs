### Path
<ParamField path="ledger_id" type="string">
  Unique `ledger_id` of the ledger being retrieved. Leave empty to view all ledgers in your Blnk Ledger.
</ParamField>

{/* <Tip>
When listing all ledgers, you can use [advanced filter operators](/search/db/filtering) to narrow down results by name, metadata, creation date, and more.
</Tip> */}

<RequestExample>
```json
curl --request GET \
  --url http://localhost:5001/ledgers/{ledger_id} \
  --header 'X-blnk-key: <api-key>'
```
</RequestExample>
