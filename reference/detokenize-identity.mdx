### Path
<ParamField path="identity_id" type="string">
  Specifies the unique ID of the identity being detokenized
</ParamField>

### Body
<ParamField body="fields" type="string[]">
  Specifies an array of fields to be detokenized.
</ParamField>

<RequestExample>
```bash
curl --request POST \
  --url http://localhost:5001/identities/{identity_id}/detokenize \
  --header 'X-blnk-key: <api-key>' \
  --header 'Content-Type: application/json' \
  --data '{
  "fields": [
    "FirstName", 
    "LastName", 
    "EmailAddress", 
    "PhoneNumber"
  ]
}'
```
</RequestExample>
<ResponseExample>
```json 201
{
  "fields" : {
    "firstName": "Alice",
    "lastName": "Smith",
    "emailAddress": "alice.smith@example.com",
    "phoneNumber": "+1234567890"
  }
}
```
</ResponseExample>
